근태결과 집계 및 마감

귀속월 기준으로 전체 재직 직원의 근태 데이터를 집계하고 마감 처리하는 화면입니다. 마감 처리 시 해당 귀속월의 근태·휴가 데이터 수정이 차단되며, 급여 시스템에서 마감된 데이터를 불러가는 기준이 됩니다. 순서: 휴가확정 → 근태마감 → 급여연동.

목차
화면 개요 마감 규칙 유틸 버튼 사원 목록 근무일 구분별 집계 총 근태 집계 휴일 구분별 집계 근태항목 집계 기타 근태항목 집계 전체 사원 근태 요약 휴일근무 집계 기준 설명 인정근무시간 산출 기준 연결 문서

화면 개요

근태결과 집계 및 마감 화면에 최초 진입하면 근무일 구분별 집계 섹션만 펼쳐진 상태로 표시되며, 나머지 집계 섹션(총 근태 집계, 휴일 구분별 집계, 근태항목 집계, 기타 근태항목 집계)은 접힌 상태입니다.

  • 각 섹션의 열기 아이콘을 클릭하면 해당 섹션이 펼쳐집니다.
  • 상단에는 전체 사원 근태 요약 카드가 표시됩니다.
  • 좌측에 사원 목록, 우측에 선택된 사원의 집계 데이터가 표시됩니다.

마감 규칙

마감 처리는 근태 데이터의 확정 상태를 결정하며, 급여 시스템에서 데이터를 불러가는 기준점이 됩니다. 아래 규칙을 반드시 준수해야 합니다.

마감 순서

마감은 과거 → 미래 순차 처리합니다.
전월이 미마감 상태인 경우 당월 마감이 불가하며, "전월 마감을 먼저 처리해주세요" 얼럿이 노출됩니다.
마감 해제는 반대로 최근 → 과거 역순으로만 가능합니다.

마감 처리 효과

마감 처리 시 해당 귀속월의 근태·휴가 데이터 수정 및 삭제가 불가합니다.
수정이 필요한 경우 마감 해제 → 각 화면에서 수정 → 재마감 순서로 처리합니다.
  • 월 마감 시 일마감은 자동으로 처리됩니다 (별도 일마감 불필요).
  • 마감 처리 후 급여 시스템에서 근태 불러오기를 통해 마감된 데이터를 전송받습니다.
  • 급여에서 불러오는 시점의 마감된 데이터가 급여에 반영됩니다.

유틸 버튼

화면 상단에 위치한 유틸 버튼들로 데이터 조회, 마감 처리, 데이터 업데이트, 엑셀 업로드/다운로드 등의 기능을 수행합니다.

귀속월

조회 대상 월을 선택합니다. 선택한 귀속월 기준으로 재직 중인 전체 사원의 근태 집계 데이터를 조회합니다.

마감 / 마감해제

버튼 토글 방식으로 마감 처리 또는 마감 해제를 수행합니다. 마감 처리 시 급여에서 근태 불러오기를 통해 데이터가 전송됩니다.

데이터 불러오기

현재 시점 기준으로 근태 및 휴가확정 데이터를 업데이트합니다.

업로드

엑셀 업로드 팝업을 호출합니다. 양식 다운로드 / 파일찾기 / 등록 / 실패내역 확인이 가능합니다.

다운로드

선택 사원의 그리드 데이터를 엑셀 파일로 다운로드합니다.

검색

사원명, 부서, 근로제 조건으로 사원 목록을 필터링하여 조회합니다.

데이터 불러오기 상세

  • 미래 날짜에서 전월 귀속월 불러오기: 해당 귀속월의 1일부터 말일까지의 데이터만 가져옵니다.
  • 월 중간에 여러 번 불러오기: 현재 시점 기준 데이터로 업데이트됩니다 (기존 데이터 덮어쓰기). 덮어쓰기 전 확인(컨펌) 팝업이 노출됩니다.
데이터 불러오기는 마감 전에 여러 번 실행할 수 있으며, 실행할 때마다 현재 시점의 최신 근태·휴가 데이터로 갱신됩니다.

사원 목록

해당 귀속월 기준으로 재직 중인 전체 사원 리스트가 표시됩니다. 사원 수가 많을 경우 스크롤 처리되며, 사원을 선택하면 우측 집계 영역에 해당 사원의 상세 근태 집계가 표시됩니다.

근무일 구분별 집계

연장근로 한도 체크가 주목적인 핵심 집계 영역입니다. 각 사원의 근로제별 근무 유형에 따라 일수와 시간을 분리 집계합니다.

번호항목설명
[1] 정상근무 각 사원의 근로제별 정상 근무시간 및 일수를 집계합니다.
[2] 조기근무 신청근로시간보다 일찍 출근한 경우에 해당합니다. 연장근무와 동일한 계산식을 적용합니다.
[3] 고정OT 포괄임금제 적용 대상입니다 (향후 지원 예정).
[4] 연장근무 근로제 시간 외 추가근무에 대한 일수 및 시간을 집계합니다.
[5] 총연장근무 [2] 조기근무 + [4] 연장근무의 합계입니다.
[6] 야간근무 22시~06시 사이의 근무 시간을 집계합니다. 자정을 넘기는 경우 전일 출근일 기준으로 집계합니다. 교대근무 시 정상근무와 야간근무를 분리 집계합니다.
[7] 휴일정상근무 주휴일·유급휴일·무급휴일·휴무일에 근로제 시간 기준 내 근무한 시간을 집계합니다.
[8] 연장_일8시간이내 휴일 초과근무 중 일 8시간 미만인 부분을 집계합니다.
[9] 연장_일8시간초과 휴일 초과근무 중 일 8시간을 초과하는 부분을 집계합니다.
[10] 연장_야간근무 휴일 초과근무 중 22시~06시에 해당하는 부분을 집계합니다.

총 근태 집계

근무일 구분별 집계를 종합하여 평일·휴일·연장·야간 기준으로 재분류한 집계입니다.

코드항목산출 기준
[A] 평일정상근무 [1] 정상근무
[B] 휴일정상근무 [7] 휴일정상근무
[C] 연장근무 (평일+휴일) [5] 총연장근무 + [8] 연장_일8시간이내 + [9] 연장_일8시간초과
[D] 야간근무 (평일+휴일) [6] 야간근무 + [10] 연장_야간근무
[F] 합계 일수 = [A] + [B] + [C]
시간 = [A] + [B] + [C] + [D]

휴일 구분별 집계

휴일수당 정산 및 대체휴무 판단이 주목적인 집계 영역입니다. 각 근로자의 근로제 타입에 따라 휴일 유형을 분리 집계합니다.

번호항목설명
[11] 주휴일근무 근로제 내 설정된 주휴일에 근무한 경우입니다. 주휴일과 공휴일이 겹치면 주휴일로 집계합니다.
[12] 유휴일근무 연휴 및 공휴일에 근무한 경우입니다.
[13] 무휴일근무 근무일·주휴일이 아닌 요일에 근무한 경우입니다.
[14] 휴무일근무 근로제에 따른 휴무일에 근무한 경우입니다.

휴일 구분별 소계

[H]유휴일[12] 유휴일근무

기타 근태항목 집계

연차 관련 집계 항목입니다.

사용연차 휴가정산에서 선택한 귀속년도 + 급여반영월 기준으로 사용한 연차 개수를 집계합니다.
정산연차 이월 제외 정산대상 연차입니다. 연차정산 = 통상일급 × 잔여연차일수로 산출합니다.

휴일근무 집계 기준 설명

휴일근무 집계는 단순히 달력상 휴일 날짜 개념이 아니라, 각 근로자의 근로제 타입에 따라 분리 집계됩니다.

근로제 타입에 따른 휴일 판단 예시
수~일 근무 근로제 적용 근로자가 토요일·일요일에 근무하더라도, 해당 요일이 근로제상 소정근로일이므로 휴일근무가 아닙니다.
공휴일 + 근로제 시간 초과 시
공휴일에 근무하면서 근로제 소정근로시간을 초과한 경우, 초과분은 휴일연장근무로 분리 집계됩니다.
야간근무 분리
22시~24시(자정) 사이의 근무시간은 휴일근무와 별도로 야간근무로 분리 집계됩니다.

인정근무시간 산출 기준

인정근무시간은 아래 기준에 따라 산출됩니다.

인정근무시간 산출 공식

1. 사원별 근로제에 따른 소정근로시간을 기준으로 실제 출퇴근시간을 합산합니다.
2. 근무패턴 내 근무시간 계산기준 설정값을 반영합니다.
3. 근로제 설정 내 휴게시간을 제외합니다.
4. 기초설정 내 연장근무 인정방식(전자결재 / 자동인정 / 미사용)에 따라 연장근무를 합산합니다.
5. 법정최소휴게시간 제외를 반영합니다.
인정근무시간의 상세 계산 로직은 인정근로시간 월마감 화면에서 사원별·일별 단위로 확인할 수 있습니다.
