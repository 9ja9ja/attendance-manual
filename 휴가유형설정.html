휴가유형 설정

회사에서 운영하는 휴가의 종류를 등록·수정·삭제하는 화면입니다. 휴가유형(연차/경조/출산육아/기타), 유무급 구분, 차감 방식(일수/시간/미차감), 유효기간, 사용여부를 설정합니다.

경로: 근태관리 > 휴가 > 휴가 설정 > 휴가유형설정 권한: 관리자

화면 개요

화면은 좌측 휴가유형 목록과 우측 상세내용으로 구성됩니다. 좌측 목록에서 항목을 선택하면 우측에 해당 유형의 상세 설정이 표시됩니다.

좌측: 휴가유형 목록

등록된 모든 휴가유형을 목록 형태로 보여줍니다. 유형별로 정렬되어 있으며, 클릭하면 우측에 상세 정보가 로드됩니다.

우측: 상세내용

선택한 휴가유형의 세부 설정값을 확인하고 수정할 수 있는 영역입니다. 기본 상태에서는 읽기 전용이며, 수정 버튼 클릭 시 활성화됩니다.

휴가유형 목록

좌측 목록에 표시되는 컬럼과 정렬 기준입니다.

컬럼설명
휴가유형연차휴가 / 경조휴가 / 출산및육아휴직 / 기타휴가
휴가구분(휴가명)각 유형 내에서 등록한 구체적인 휴가 이름
차감일수해당 휴가 사용 시 차감되는 일수
소요시간시간 단위 차감 시 소요시간
사용여부사용 / 미사용 상태
정렬 순서: 연차휴가 → 경조휴가 → 출산및육아휴직 → 기타휴가 순으로 표시되며, 각 유형 내에서는 등록 순서대로 나열됩니다.

상세내용

우측에 표시되는 상세 필드입니다. 목록에서 항목을 선택하면 아래 필드에 값이 로드됩니다.

휴가유형 드롭다운 선택. 연차휴가(default) / 경조휴가 / 출산및육아휴직 / 기타휴가. 최초 저장 후에는 수정할 수 없습니다.
휴가명 직접 입력. 최대 50자까지 입력 가능합니다. 동일 휴가유형 내에서 중복된 이름은 등록할 수 없습니다.
유/무급구분 유급(default) / 무급. 유급 휴가만 연차정산 시 잔여건수에 노출됩니다.
휴가소진구분 연차휴가 유형을 선택한 경우 비활성화됩니다. 그 외 유형에서만 활성화되어 선택할 수 있습니다.
휴가차감일수 일수(default) / 시간 / 미차감
· 일수: 소요시간 필드가 비활성화됩니다.
· 시간: 휴가기간 필드가 비활성화됩니다.
· 미차감: 소요시간 + 휴가기간 필드가 모두 비활성화됩니다.
유효기간 사용여부 사용(default) / 미사용. 미사용으로 설정하면 기간설정 필드가 비활성화됩니다.
유효기간 기간설정 유효기간 사용여부가 "사용"일 때 활성화됩니다.
· 회계연도 말일까지: 해당 연도 12월 31일에 자동 소멸
· 발생일 기준: 발생일로부터 N일 / N개월 / N년 후 소멸
· 직접설정: 달력에서 특정 날짜를 지정
사용여부 사용(default) / 미사용. 미사용으로 전환하면 직원 신청 화면에서 해당 휴가가 노출되지 않습니다.

주요 버튼

등록

새로운 휴가유형을 등록하는 팝업을 엽니다.

수정

선택한 항목의 상세내용 필드를 활성화하여 수정할 수 있습니다. 단, 휴가유형은 수정할 수 없습니다.

삭제

선택한 항목을 삭제합니다. 전자결재에서 사용 중인 유형은 삭제할 수 없습니다.

취소

수정 중인 내용을 취소하고 원래 값으로 되돌립니다.

저장

수정한 상세내용을 저장합니다. 저장 전 필수 항목 누락 시 얼럿이 표시됩니다.

등록 팝업

등록 버튼 클릭 시 표시되는 팝업입니다. 상세내용과 동일한 필드가 제공됩니다.

연차휴가 선택 시 제한사항

휴가유형을 "연차휴가"로 선택하면 아래 항목이 자동 고정됩니다.
· 유/무급구분: 유급 고정
· 휴가차감일수: 일수 고정
· 유효기간 사용여부: 미사용 고정

등록 시 규칙

  • 필수 입력: 휴가명은 반드시 입력해야 합니다. 미입력 시 "휴가명을 입력해주세요." 얼럿이 표시됩니다.
  • 중복 검사: 동일 휴가유형 내에서 같은 휴가명이 이미 존재하면 "이미 등록된 휴가명입니다." 얼럿이 표시됩니다.
  • 최초 저장 후: 휴가유형 필드는 수정할 수 없습니다. 유형을 변경하려면 삭제 후 재등록해야 합니다.
등록 팝업에서 저장하면 좌측 목록에 즉시 반영됩니다. 취소 시 입력 내용은 모두 초기화됩니다.

삭제 운영 규칙

휴가유형 삭제 시 전자결재 연동 여부에 따라 동작이 달라집니다.

전자결재 미사용 유형

삭제 확인 얼럿 후 즉시 삭제됩니다. 삭제된 유형으로 이미 신청된 휴가 이력은 유지되지만, 이후 신규 신청은 불가합니다.

전자결재 사용 중인 유형

삭제할 수 없습니다. "전자결재에서 사용 중인 휴가유형은 삭제할 수 없습니다." 얼럿이 표시됩니다. 대신 사용여부를 "미사용"으로 전환하여 신규 신청을 차단할 수 있습니다.

전자결재에서 사용 중인 유형을 비활성화해야 하는 경우, 먼저 전자결재 서비스에서 해당 양식의 사용을 중지한 뒤 삭제하거나, 사용여부를 "미사용"으로 변경하십시오.
