근무지 설정 화면은 출퇴근 인정 기준을 실제로 만드는 입력 화면입니다

근무지 설정은 주소를 입력하는 화면이 아니라, 직원이 어디에서 어떤 방식으로 출퇴근을 인정받을 수 있는지를 결정하는 화면입니다. GPS, WEB IP, WIFI 방식 선택에 따라 허용 범위가 달라지고, 대표 근무지 여부에 따라 신규 직원 기본 배포 기준도 달라집니다. 따라서 이 화면은 입력 화면이면서 동시에 출퇴근 인정 정책을 구성하는 화면으로 이해해야 합니다.

설정 화면 개요

근무지 설정 화면은 근무지 목록에서 등록 또는 수정을 선택했을 때 열리는 상세 입력 화면입니다. 이 화면에서 입력한 값은 직원 출퇴근 허용 장소, 허용 네트워크, GPS 반경 기준, 신규 직원 기본 근무지 기준으로 이어집니다. 따라서 정확한 값 입력과 저장 전 검증이 중요합니다.

등록 화면의 역할

새 근무지를 만들 때 필요한 기본 정보와 출퇴근 인정 방식을 입력합니다. 사무실, 지점, 현장, 상주 장소처럼 근무 형태가 다른 곳도 같은 흐름 안에서 관리합니다.

수정 화면의 역할

이미 등록된 근무지의 주소, 반경, IP, WIFI, 대표 여부를 변경합니다. 이미 직원에게 배포된 근무지일 수 있으므로 변경 영향도 함께 검토해야 합니다.

출퇴근 체크 방식 설정 로직

출퇴근 체크 방식은 이 근무지에서 어떤 조건으로 타각을 인정할 것인지 정하는 핵심 설정입니다. 화면에 처음 들어왔을 때는 관련 입력 영역이 비활성 상태로 보이고, GPS, WEB IP, WIFI를 선택하면 각 방식에 맞는 세부 입력 영역이 열립니다.

GPS

좌표와 반경을 기준으로 특정 위치 안에서만 출퇴근을 허용합니다. 지점, 현장, 사옥처럼 물리적 위치가 분명한 장소에 적합합니다.

WEB IP

등록된 IP 대역 안에서만 출퇴근을 허용합니다. 고정 회선을 사용하는 사무실 환경에 적합합니다.

WIFI

등록된 WIFI 환경 안에서만 출퇴근을 허용합니다. 모바일 출퇴근이나 현장 무선 네트워크 환경 관리에 유용합니다.

체크 방식은 단순 설명 문구가 아니라 직원 화면에 그대로 반영되는 운영 기준입니다. 어떤 방식이 실제 사용 환경에 맞는지 먼저 정한 뒤 세부값을 입력해야 합니다.

주소 / 좌표 / 반경 설정

주소

주소는 우편번호와 기본 주소, 상세주소로 나누어 관리합니다. 주소 검색을 통해 기준 주소를 불러오고, 필요한 경우 상세주소를 추가합니다.

  • 주소 검색 기능을 통해 기준 주소를 선택합니다.
  • 기본 주소와 상세주소를 구분해 관리합니다.
  • 주소가 정해지면 지도와 좌표 기준도 함께 정리됩니다.

좌표

좌표는 주소 검색 결과나 지도 마커 선택에 따라 자동으로 정해집니다. 숫자를 직접 손으로 입력하기보다, 위치 선택 결과를 기준으로 관리하는 방식이 더 자연스럽습니다.

  • 주소 또는 지도 선택 결과로 좌표가 자동 반영됩니다.
  • 마커를 이동하면 위치 기준이 갱신됩니다.
  • 주소가 없는 상태에서는 브라우저 위치 기준이 사용될 수 있습니다.

반경

항목설정 기준설명
기본값50최초 입력 시 제시되는 기본 반경입니다.
입력 형식숫자만 입력반경은 숫자 기반 값으로 관리합니다.

지도 마커를 옮기면 좌표와 주소가 다시 정리될 수 있습니다. 다만 상세주소는 운영 메모 성격이 강하므로 별도 확인이 필요합니다.

WEB IP 설정

WEB IP는 특정 네트워크 대역 안에서만 출퇴근을 허용하는 방식입니다. 일반적으로 사무실 PC 환경처럼 네트워크가 비교적 고정된 경우에 적합합니다.

  • 점으로 구분된 IP 형식인지 검증합니다.
  • 각 자리 수는 0~255 범위인지 확인합니다.
  • 현재 접속 중인 IP를 자동으로 입력할 수 있습니다.
  • 여러 대역이 필요하면 행을 추가해 다중 등록합니다.
WEB IP는 관리가 편리하지만 회선 교체나 사무실 이전이 있으면 값 정비가 필요합니다. 네트워크 변경 시 출퇴근이 갑자기 실패할 수 있기 때문입니다.

WIFI 설정

WIFI는 특정 무선 네트워크 환경 안에서만 출퇴근을 허용하는 방식입니다. 현장, 이동 근무지, 소규모 지점처럼 모바일 출퇴근 비중이 높은 경우에 유용합니다.

  • 허용 형식에 맞는 WIFI 정보인지 검증합니다.
  • 현재 접속 중인 WIFI 정보를 자동으로 채울 수 있습니다.
  • 필요하면 여러 행으로 추가 등록합니다.
WIFI는 현장 사정에 따라 자주 바뀔 수 있으므로, 실제 사용 환경과 등록 정보가 일치하는지 주기적으로 점검하는 것이 좋습니다.

대표 근무지 설정 규칙

대표 근무지는 신규 직원에게 기본으로 배포되는 기준 근무지입니다. 따라서 단순 체크 항목이 아니라 회사의 기본 출퇴근 기준을 정하는 설정입니다.

  • 최초 등록 단계에서는 대표 근무지가 반드시 존재해야 합니다.
  • 기존 대표 근무지를 다른 근무지로 바꿀 때는 확인 메시지를 거칩니다.
  • 대표 근무지 변경은 신규 배포 기준 변경과 연결되므로 신중한 검토가 필요합니다.
확인대표 근무지를 변경하시겠습니까.

운영 의미: 대표 근무지는 신규 입사자 기본값과 연결되므로, 실수로 잘못 바꾸지 않도록 다시 한번 확인합니다.

직원 출퇴근 적용 영향

근무지 설정값은 저장 후 직원의 출퇴근 인정 방식에 직접 영향을 줍니다. 따라서 이 화면은 관리자용 입력 화면이면서, 동시에 직원 출퇴근 정책을 정하는 화면이기도 합니다.
