연차촉진 현황

연차촉진 관리에서 진행한 1차·2차 촉진의 진행 상태를 모니터링하는 화면입니다. 촉진 대상 여부, 통보 상태, 제출 현황을 확인하고, 직원별 연차계획 회신 내역을 조회할 수 있습니다.

경로: 근태관리 > 연차 > 연차촉진 현황 권한: 관리자

화면 개요

최초 진입 시 default 조건은 구분: 1년 이상, 기간: 당해년도 1월 1일 ~ 12월 31일입니다.

이 화면은 연차촉진 관리에서 실행한 촉진 절차의 결과를 확인하는 모니터링 전용 화면입니다. 촉진 통보 발송은 연차촉진 관리 화면에서 수행합니다.

검색 영역

구분 1년 이상 / 1년 미만을 선택합니다. 근속 기간에 따라 촉진 기준이 다르므로 탭이 분리되어 있습니다.
검색 사원명 또는 부서명으로 like 검색합니다.
촉진기준일 기간 범위를 설정하여 해당 기간 내 촉진기준일에 해당하는 사원을 조회합니다.
촉진구분 전체(default) / 1차 촉진 / 2차 촉진을 선택합니다.
상태 대기 / 진행중 / 제출완료 / 미제출 상태별로 필터링합니다.

진행 단계 요약

화면 상단에 촉진 절차의 전체 진행 상황을 4단계 카드 형태로 요약하여 보여줍니다.

[1단계] 대상자 확정

전체 N명 중 N명 확정
미확정 N명

[2단계] 1차 촉진(알림)

발송 완료 N명
미확인 N명

[3단계] 계획 접수

제출 N명
미제출 N명

[4단계] 2차 촉진(지정)

대상 N명
(활성화 시점 표시)

미확정 인원은 빨간색으로 강조 표시됩니다. 각 단계의 수치는 검색 조건과 관계없이 전체 현황을 기준으로 집계됩니다.

현황 리스트

검색 조건에 따라 촉진 대상자의 상세 정보가 테이블 형태로 표시됩니다.

구분 1년 이상 또는 1년 미만을 표시합니다.
상태 대기 / 진행중 / 제출완료 / 미제출 상태를 표시합니다.
부서 / 사원 / 직급 사원의 소속 부서, 사원명, 직급 정보입니다.
입사일 사원의 입사일을 표시합니다.
재직구분 재직 또는 퇴직 상태입니다.
촉진대상 잔여일수가 있으면 "여", 없으면 "부"로 표시됩니다.
촉진기준일 자동 계산된 촉진 기준일입니다. 1년 이상과 1년 미만에 따라 각각 다른 기준으로 산출됩니다.
촉진구분 1차 촉진 또는 2차 촉진 구분을 표시합니다.

연차계획보기 팝업

현황 리스트에서 사원의 상태를 더블클릭하면 연차계획 회신 내역을 확인할 수 있는 팝업이 표시됩니다.

  • 직원이 1차 촉진 메일을 통해 등록한 연차 사용 계획 내역을 확인할 수 있습니다.
  • 회신일자, 계획한 연차 사용일, 사용 일수 등이 표시됩니다.
  • 제출완료 상태인 사원만 상세 내역이 조회됩니다.
팝업에서는 조회만 가능하며 수정은 불가합니다. 연차계획의 등록·수정은 직원 본인이 촉진 메일의 링크를 통해 수행합니다.

연차 사용시기 지정 설정

2차 촉진 시 관리자가 직접 미사용 연차의 사용시기를 지정하는 기능입니다. 1차 촉진에서 미제출 상태인 사원에 대해 관리자가 특정 일자를 휴가로 지정합니다.

적용요일 선택

연차를 지정할 요일을 선택합니다. 선택한 요일에 해당하는 날짜가 자동으로 연차 사용일로 배정됩니다.

잔여연차 대비 적용요일 부족

선택한 적용요일의 일수가 잔여연차 개수보다 적을 경우 경고 알럿이 표시됩니다. 추가 요일을 선택하거나 기간을 조정해야 합니다.

개수 초과

지정한 연차 개수가 잔여연차를 초과할 경우 경고 알럿이 표시됩니다. 초과된 일수를 조정해야 합니다.

2차 촉진에서 사용시기를 지정하면 해당 일자는 확정된 휴가일로 처리됩니다. 지정 전에 반드시 잔여연차 일수와 적용요일을 확인하십시오.
