간주근무 유형 관리

사업장 밖 근로 등 근로시간 산정이 어려운 경우(재택, 장기 출장 등)에 사용하는 간주근무 유형을 등록·수정·삭제하는 화면입니다. 간주근무는 소정근무시간을 근무한 것으로 인정하므로 근무 인정이 항상 "인정" 상태입니다.

경로: 근태관리 > 근무유형 > 근태 유형 설정 > 간주근무 권한: 관리자
목차
화면 개요 검색 영역 유형 리스트 상세내용 주요 버튼 등록 팝업 삭제·수정 운영 규칙 다른 유형과의 차이점 연결 문서

화면 개요

간주근무 유형 관리 화면은 검색 + 리스트 + 상세 3단 구조로 구성됩니다. 다른 유형 관리 화면과 동일한 레이아웃이지만, 간주근무 특성상 필드 수가 가장 적고 단순한 구조입니다.

검색 영역

간주근무명, 사용여부 등 조건으로 등록된 유형을 조회합니다.

리스트 영역

등록된 간주근무 유형을 표 형태로 표시합니다.

상세 영역

선택한 유형의 근무 인정 시간, 휴게 인정 시간 등 세부 정보를 확인·수정합니다.

검색 영역

검색 조건은 기본적으로 공란(전체 조회) 상태입니다. 별도 조건을 입력하지 않고 조회하면 등록된 모든 간주근무 유형이 리스트에 표시됩니다.

유형 리스트

조회 결과는 아래 컬럼으로 구성된 테이블에 표시됩니다.

컬럼 설명
간주근무명 관리자가 등록한 간주근무 유형의 명칭입니다.
근무인정시간 해당 유형으로 인정하는 근무 시간입니다.
휴게인정시간 해당 유형에 설정된 휴게 시간입니다.
사용여부 사용 또는 미사용 상태를 표시합니다.

상세내용

리스트에서 유형을 선택하면 하단 상세 영역에 아래 필드가 표시됩니다.

간주근무명 명칭을 입력합니다. 직원 신청 페이지에서 선택 항목으로 노출되며, 전자결재에서도 구분 값으로 사용됩니다.
근무 인정 시간 간주 근무로 인정하는 근무 시간을 입력합니다. 해당 유형으로 신청이 승인되면 이 시간이 인정근로시간으로 반영됩니다.
휴게 인정 시간 휴게 시간을 입력합니다. 법정 기준에 따라 4시간 근무 시 30분 이상, 8시간 근무 시 1시간 이상의 휴게 시간을 설정해야 합니다.
근무 인정 항상 "인정" 고정입니다. 변경할 수 없습니다.
사용여부 사용(default) / 미사용 중 선택합니다. 미사용으로 전환하면 직원 신청 화면에서 해당 유형이 노출되지 않습니다. 기존 데이터는 유지됩니다.
간주 근무제는 소정근무 시간을 근무하였다고 인정하는 근무 유형이므로 인정 상태로만 사용이 가능합니다. "미인정"으로 전환할 수 없으며, 근무 인정 필드는 비활성화(읽기 전용) 상태로 표시됩니다.

주요 버튼

등록 새로운 간주근무 유형을 등록하는 팝업을 엽니다.
수정 리스트에서 선택한 유형의 상세 정보를 수정 모드로 전환합니다.
삭제 선택한 유형을 삭제합니다. 전자결재 연동 여부에 따라 삭제 가능 여부가 결정됩니다.
취소 수정 중인 내용을 저장하지 않고 원래 상태로 되돌립니다.
저장 수정한 상세 정보를 저장합니다.

등록 팝업

등록 버튼을 클릭하면 팝업이 열리며, 상세내용과 동일한 필드가 표시됩니다. 추가로 비고 필드가 포함되어 운영 메모를 남길 수 있습니다.

간주근무명 유형 명칭 입력
근무 인정 시간 시간 입력
휴게 인정 시간 시간 입력
근무 인정 인정 (고정, 변경 불가)
사용여부 사용(default) / 미사용
비고 운영 메모, 적용 대상 부서 등 자유 입력
등록 완료 후 리스트에 즉시 반영됩니다. 전자결재 연동이 활성화된 경우 등록된 유형이 전자결재 양식의 선택 항목에도 반영됩니다.

삭제·수정 운영 규칙

간주근무 유형의 삭제·수정은 전자결재 연동 여부에 따라 처리 방식이 달라집니다.

전자결재 미사용 시

유형을 자유롭게 수정·삭제할 수 있습니다. 이미 승인된 신청 데이터가 있는 경우에도 삭제가 가능하지만, 기존 데이터의 유형 참조가 끊어질 수 있으므로 "미사용" 전환을 권장합니다.

전자결재 사용 시

진행 중인 결재 건이 있는 유형은 삭제할 수 없습니다. 결재가 완료되거나 반려된 후에만 삭제가 가능합니다. 수정 시에도 진행 중인 결재 건에는 영향을 주지 않으며, 이후 신규 신청부터 변경 내용이 적용됩니다.

삭제보다 사용여부를 "미사용"으로 전환하는 것이 안전합니다. 과거 근태 이력에서 유형 정보를 조회할 수 있도록 데이터를 유지할 수 있습니다.

다른 유형과의 차이점

간주근무 유형 관리는 외근·출장·연장근무와 비교하여 아래 세 가지 고유한 특징이 있습니다.

간주근무만의 특징 3가지
1. 항상 인정 간주근무는 소정근무시간을 근무한 것으로 보는 제도이므로, 근무 인정이 항상 "인정"으로 고정됩니다. 외근·출장·연장근무처럼 인정/미인정을 선택할 수 없습니다.
2. 출퇴근 인정 없음 근로시간 산정이 어려운 상황을 전제하므로, 출/퇴근 인정 여부 설정 자체가 존재하지 않습니다. 외근·출장에는 있는 출/퇴근 인정 필드가 간주근무에는 없습니다.
3. 근태유형 선택 없음 연장근무는 연장/야간/휴일 중 근태유형을 선택하고, 외근·출장은 고정 유형이 있습니다. 간주근무는 근태유형 선택 드롭다운 없이 간주근무명만 입력합니다.
