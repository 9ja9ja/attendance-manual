나의 근태 현황

직원이 배포받은 근로제·근무지 정보를 조회하고, 출퇴근을 등록하며, 주간/월간 근무시간 현황과 근태 신청 내역을 확인하는 직원 전용 화면입니다.

목차
화면 개요 나의 근로 정보 영역 나의 근로 정보 팝업 근로제 수정 요청 팝업 출퇴근 버튼 근무 시간 영역 근태 신청 내역 연결 문서

화면 개요

나의 근태 현황은 직원이 로그인 후 진입하는 첫 화면입니다. 관리자가 근로제를 배포하면 해당 직원의 근로 정보, 출퇴근 등록 버튼, 근무시간 현황, 근태 신청 내역이 한 화면에 통합 표시됩니다.

근로제 배포 완료

나의 근로 정보, 출퇴근 버튼, 근무시간 영역, 근태 신청 내역이 모두 정상 조회됩니다.

근로제 미배포

"이용 가능한 근로제 정보가 없습니다" 안내 메시지가 표시되며, 출퇴근 등록 및 근태 신청 기능을 사용할 수 없습니다.

근로제가 배포되지 않은 직원은 화면의 모든 기능이 비활성화됩니다. 관리자에게 근로제 배포를 요청해야 합니다.

나의 근로 정보 영역

배포받은 근로제와 근무지 기반의 근로 정보를 요약하여 보여주는 영역입니다.

나의 근로제 관리자가 배포한 근로제명을 조회합니다. 근로제명을 선택하면 나의 근로 정보 팝업이 열려 상세 정보를 확인할 수 있습니다.
나의 근로 정보 팝업 근로제명 옆 안내 아이콘(i)을 선택하면 근로제 상세 정보를 팝업으로 표시합니다. 근로제 유형에 따라 표시 항목이 달라집니다.
근로제 수정 요청 관리자가 근로제 설정에서 수정 요청을 허용한 경우에만 노출됩니다. 직원이 근로제 변경을 관리자에게 요청할 수 있습니다.
나의 근무지 배포받은 근무지명을 조회합니다. 2곳 이상인 경우 마우스 오버(hover) 시 도움말로 전체 근무지 목록이 표시됩니다.
출퇴근 인정 방식 배포된 근로제에 설정된 출퇴근 인정 정보를 전체 노출합니다. GPS, WEB IP, WIFI 등 설정된 인정 방식이 모두 표시됩니다.
1주 소정 근로 시간 배포된 근로제에 설정된 주간 소정근로시간을 조회합니다.
이번 달 소정 근로일 시스템 월 기준으로 계산된 소정근로일수를 표시합니다. 주휴일, 공휴일 등을 제외한 근무해야 할 일수입니다.
이번 달 소정 근로 시간 1주 소정근로시간 × 이번 달 소정근로일로 계산된 월간 소정근로시간을 표시합니다.
나의 부여 휴가 현재 회기 기준으로 부여된 휴가의 합계를 표시합니다.
나의 사용 휴가 현재 회기 기준으로 사용(소진)한 휴가의 합계를 표시합니다.
나의 잔여 휴가 부여 휴가에서 사용 휴가를 차감한 잔여 휴가 합계를 표시합니다.
이번달 근태 이상 해당 월의 미퇴근/결근/지각/조퇴 총 건수를 표시합니다. 건수를 선택하면 출퇴근 조회 화면으로 이동합니다.
이번달 근태 신청 해당 월의 전체 근태 신청 건수를 표시합니다. 건수를 선택하면 하단의 근태 신청 내역 영역으로 스크롤 이동합니다.

나의 근로 정보 팝업

근로제 유형에 따라 팝업에 표시되는 항목이 달라집니다. 표준/시차/선택 근로제와 스케줄 근로제 두 가지로 구분됩니다.

공통 항목

근로제명배포받은 근로제의 이름을 표시합니다.
근로제 유형표준/시차/선택/스케줄 등 근로제 유형을 표시합니다.
근무지명배포받은 근무지의 이름을 표시합니다.
연장근무 인정 방식근로제에 설정된 연장근무 인정 방식을 표시합니다.
출퇴근 체크 방식GPS, WEB IP, WIFI 등 설정된 출퇴근 체크 방식을 표시합니다.
확인 버튼팝업을 닫습니다.

표준 / 시차 / 선택 근로제

근무요일 — 주간 근무 요일
주휴요일 — 주휴일로 지정된 요일
표준근무시간 — 일일 표준 근무 시작~종료 시간
표준휴게시간 — 일일 표준 휴게 시작~종료 시간

스케줄 근로제

근무일수 — 스케줄 주기 내 근무일 수
출근일자 — 스케줄 기준 출근 대상 일자
휴무일자 — 스케줄 기준 휴무 대상 일자
등록근무시간 — 스케줄에 등록된 근무 시간
등록휴게시간 — 스케줄에 등록된 휴게 시간

근로제 수정 요청 팝업

관리자가 근로제 설정에서 수정 요청을 허용한 경우, 직원이 근로제 변경을 요청할 수 있는 팝업입니다.

입력 항목

기존 근로제 현재 배포받은 근로제명을 조회 전용으로 표시합니다. 수정할 수 없습니다.
수정 근로제 유형 사용 상태인 근로제 유형을 드롭다운으로 선택합니다.
수정 근무제 리스트 선택한 근로제 유형 내에서 사용 상태인 근로제명을 드롭다운으로 선택합니다.
수정 근무 요일 선택한 근로제의 근무 요일 정보가 자동 조회됩니다.
수정 근무 시간 선택한 근로제의 근무 시간 정보가 자동 조회됩니다.
요청 사유 최대 30자 입력. 필수 입력 항목입니다.
적용 요청 일자 기본값은 시스템일자 + 1일입니다. 과거 일자는 선택할 수 없습니다.

요청 처리

성공 "정상적으로 관리자에게 요청되었습니다."
요청 버튼을 선택하면 관리자에게 근로제 수정 요청이 전달됩니다. 요청 사유와 적용 요청 일자를 정확하게 입력해야 합니다.

수정 요청 대기 상태

이미 수정 요청을 보낸 상태에서 다시 근로제 수정 요청 아이콘을 선택하면, 기존 요청 내역을 조회 팝업으로 표시합니다.

  • 기존 요청의 상세 정보(수정 근로제, 적용 일자, 요청 사유 등)를 확인할 수 있습니다.
  • 요청 취소 버튼을 통해 대기 중인 요청을 취소할 수 있습니다.
  • 요청이 취소되면 다시 새로운 수정 요청을 보낼 수 있습니다.

출퇴근 버튼

직원이 출근/퇴근 시간을 직접 등록하는 버튼입니다. 자동 시간 입력이 아닌 근로제에서만 활성화됩니다.

버튼 활성화 규칙

출근 버튼

출근 미등록 상태에서 출근 버튼만 활성화됩니다. 퇴근 버튼은 비활성 상태입니다.

퇴근 버튼

출근 등록 후 퇴근 버튼이 활성화됩니다. 퇴근 버튼은 계속 활성 상태이며, 여러 번 등록해도 마지막 시점으로 저장됩니다.

출퇴근 등록 컨펌

출근 "%시간% 출근 시간으로 등록하시겠습니까?"
퇴근 "%시간% 퇴근 시간으로 등록하시겠습니까?"

출퇴근 인정 방식 불일치 얼럿

출퇴근 등록 시 배포된 근로제의 출퇴근 인정 방식과 현재 환경이 일치하지 않으면 아래 얼럿이 표시되며 등록이 차단됩니다.

GPS "등록된 근무지 주소와 멀리 떨어져 있어 정상적으로 등록되지 않습니다."
WEB IP "등록된 WEB IP 대역과 맞지 않아 정상적으로 등록되지 않습니다."
WIFI "등록된 WIFI 주소와 맞지 않아 정상적으로 등록되지 않습니다."
출퇴근 인정 방식에 따른 검증이 실패하면 출퇴근 등록이 정상적으로 처리되지 않습니다. 등록된 근무지에서 출퇴근을 진행해야 합니다.

근무 시간 영역

주간 또는 월간 단위로 근무시간 현황을 조회하는 영역입니다.

조회 구분 탭

주간 (기본값)

시스템일자가 포함된 주(월~일)를 기준으로 일별 근무시간을 조회합니다.

월간

시스템일자의 년/월을 기준으로 월별 근무시간을 조회합니다.

기간 이동

기간 선택 주간 탭은 시스템일자 포함 주, 월간 탭은 시스템 년/월이 기본 선택됩니다. 좌/우 화살표로 이전/다음 기간으로 이동합니다.
오늘 버튼 시스템일자 기준으로 재조회합니다. 다른 기간으로 이동한 후 현재 기간으로 돌아올 때 사용합니다.
일자 이동 버튼 특정 일자를 선택하면 해당 일자의 일별 근무시간 상세를 조회합니다.

근태 이상 버튼

일별 근무시간 조회 시 근태 이상이 발생한 날에는 근태 이상 버튼이 표시됩니다. 각 유형별로 아래 컨펌 메시지가 표시됩니다.

미출근 "출근 기록이 없습니다. 출퇴근 수정 신청을 하시겠습니까?"
미퇴근 "퇴근 기록이 없습니다. 출퇴근 수정 신청을 하시겠습니까?"
결근 "결근 처리되었습니다. 출퇴근 수정 신청을 하시겠습니까?"
지각 "지각 처리되었습니다. 출퇴근 수정 신청을 하시겠습니까?"
조퇴 "조퇴 처리되었습니다. 출퇴근 수정 신청을 하시겠습니까?"
의무시간 미충족 "의무 근무 시간이 충족되지 않았습니다. 출퇴근 수정 신청을 하시겠습니까?"

일일 근무시간 그래프

일별 근무시간을 시각적 그래프로 표시합니다. 소정 근무 시간 내/외로 구분됩니다.

소정 근무 시간 내

정규 근무시간 범위 내에서의 근무를 표시합니다. 출장, 외근, 휴가, 간주 시간이 반영됩니다.

소정 근무 시간 외

정규 근무시간 초과분을 표시합니다. 연장, 야간, 휴일 근무 시간이 반영됩니다.

신청 내역

해당 일자에 등록된 근태 신청 내역을 표시합니다.

근태유형외근, 출장, 연장근무, 간주근무, 휴가 등 신청한 근태의 유형을 표시합니다.
구분신청 건의 세부 구분 정보를 표시합니다.
제목[사유]신청 시 입력한 제목과 사유를 표시합니다.

총 근무 시간 그래프

조회 기간(주간/월간) 전체의 근무시간을 종합 그래프로 표시합니다.

소정근무시간해당 기간의 소정근무시간 합계를 표시합니다.
잔여소정근무시간 중 아직 남은(미달) 시간을 표시합니다.
그 외 근무시간연장, 야간, 휴일 등 소정근무시간 외의 근무시간 합계를 표시합니다.

근태 신청 내역

직원이 신청한 근태 내역을 목록으로 조회하는 영역입니다. 최대 10개까지 표시되며, 초과 시 스크롤로 조회합니다.

컬럼 구성

컬럼 설명
근태유형외근, 출장, 연장근무, 간주근무, 휴가 등 신청 유형
근태구분유형 내 세부 구분(예: 국내출장, 야간연장 등)
시작일자근태 신청의 시작 일자
종료일자근태 신청의 종료 일자
시간신청한 시간 범위 또는 시간 수
신청일자근태를 신청한 일자
근태상태승인/반려/대기 등 근태 처리 상태
결재문서번호전자결재 연동 시 생성된 결재 문서 번호
결재상태전자결재 연동 시 결재 진행 상태(진행중/완료/반려 등)

부가 기능

  • 접기/펼치기 버튼: 근태 신청 내역 영역 전체를 접거나 펼칠 수 있습니다.
  • 전자결재 연동: 전자결재 서비스와 연동된 경우, 결재상태 컬럼이 표시됩니다. 해당 row를 선택하면 전자결재 팝업이 열려 결재 문서 상세를 확인할 수 있습니다.
전자결재 미연동 시 결재문서번호, 결재상태 컬럼은 표시되지 않으며, 근태상태만으로 처리 현황을 확인합니다.
