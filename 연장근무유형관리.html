연장근무 유형 관리

직원이 연장·야간·휴일 근무를 신청할 때 사용할 수 있는 유형을 등록·수정·삭제하는 화면입니다. 외근·출장과 달리 근태 유형을 연장근무/야간근무/휴일근무 중 선택할 수 있고, 근무 시간을 from~to로 지정합니다.

경로: 근태관리 > 근무유형 > 근태 유형 설정 > 연장근무 권한: 관리자
목차
화면 개요 검색 영역 유형 리스트 상세내용 주요 버튼 등록 팝업 삭제·수정 운영 규칙 외근·출장과의 차이점 요약 연결 문서

화면 개요

연장근무 유형 관리 화면은 검색 + 리스트 + 상세 3단 구조로 구성됩니다. 상단 검색 영역에서 조건을 입력하면 리스트에 해당 유형이 표시되고, 리스트에서 항목을 선택하면 하단 상세 영역에 세부 정보가 나타납니다.

검색 영역

유형명, 사용여부 등 조건으로 등록된 연장근무 유형을 조회합니다.

리스트 영역

등록된 연장근무 유형 목록을 표 형태로 표시합니다.

상세 영역

선택한 유형의 근태유형, 근무시간(from~to), 근무 인정 등 세부 정보를 확인·수정합니다.

검색 영역

검색 조건은 기본적으로 공란(전체 조회) 상태입니다. 별도 조건을 입력하지 않고 조회하면 등록된 모든 연장근무 유형이 리스트에 표시됩니다.

유형 리스트

조회 결과는 아래 컬럼으로 구성된 테이블에 표시됩니다.

컬럼 설명
근태유형 연장근무 / 야간근무 / 휴일근무 중 해당 유형이 표시됩니다.
연장근무구분(연장근무명) 관리자가 등록한 유형 명칭입니다.
근무인정 인정 또는 미인정 상태를 표시합니다.
근무시간 해당 유형의 근무 시작~종료 시간(from~to)을 표시합니다.
사용여부 사용 또는 미사용 상태를 표시합니다.

상세내용

리스트에서 유형을 선택하면 하단 상세 영역에 아래 필드가 표시됩니다. 이 영역은 외근·출장 유형 관리와 가장 큰 차이를 보이는 핵심 구간입니다.

근태 유형 연장근무(default) / 야간근무 / 휴일근무 드롭다운에서 선택합니다.
외근·출장은 근태 유형이 고정(외근 또는 출장)되지만, 연장근무 유형 관리에서는 세 가지 중 하나를 직접 선택할 수 있습니다.
연장근무구분
(연장근무명)		 직원이 신청 시 선택하게 될 명칭을 입력합니다. 전자결재에서도 이 명칭이 구분 값으로 사용됩니다.
근무 시간 from ~ to 형태로 시작 시간과 종료 시간을 지정합니다(default: 18:00 ~ 22:00).
30분 단위 드롭다운으로 선택하거나 직접 입력이 가능합니다.
우측에 총 근무시간(근무시간 - 휴게시간)이 자동 표시됩니다.
휴게 시간 from ~ to 형태로 휴게 시작과 종료 시간을 지정합니다. 근무 시간 구간 내에서 설정합니다.
근무 인정 시간 근무시간 - 휴게시간으로 자동 계산됩니다. 관리자가 직접 수정할 수 없습니다.
휴게 인정 시간 선택한 휴게 시간의 시작/종료 값으로 자동 계산됩니다.
근무 인정 인정(default) / 미인정 중 선택합니다.
· 인정: 해당 연장근무 실적이 근태 데이터에 반영되어 인정근로시간 계산에 포함됩니다.
· 미인정: 연장근무 실적만 기록하고, 근태 데이터(인정근로시간)에는 반영하지 않습니다.
사용여부 사용(default) / 미사용 중 선택합니다. 미사용으로 전환하면 직원 신청 화면에서 해당 유형이 노출되지 않습니다. 기존 데이터는 유지됩니다.
연장근무 유형의 핵심은 근태 유형을 연장근무·야간근무·휴일근무 중 선택하고, 근무 시간을 from~to로 구간 지정한다는 점입니다. 외근·출장처럼 총 인정 시간을 시간+분으로 입력하는 방식이 아니라, 실제 근무 구간을 시간대로 정의합니다.

주요 버튼

등록 새로운 연장근무 유형을 등록하는 팝업을 엽니다.
수정 리스트에서 선택한 유형의 상세 정보를 수정 모드로 전환합니다.
삭제 선택한 유형을 삭제합니다. 전자결재 연동 여부에 따라 삭제 가능 여부가 결정됩니다.
취소 수정 중인 내용을 저장하지 않고 원래 상태로 되돌립니다.
저장 수정한 상세 정보를 저장합니다.

등록 팝업

등록 버튼을 클릭하면 팝업이 열리며, 상세내용과 동일한 필드가 표시됩니다. 추가로 비고 필드가 포함되어 운영 메모를 남길 수 있습니다.

근태 유형 연장근무(default) / 야간근무 / 휴일근무 선택
연장근무구분 유형 명칭 입력
근무 시간 from ~ to (default: 18:00 ~ 22:00)
휴게 시간 from ~ to
근무 인정 시간 자동 계산
휴게 인정 시간 자동 계산
근무 인정 인정(default) / 미인정
사용여부 사용(default) / 미사용
비고 운영 메모, 예외 기준 등 자유 입력
등록 완료 후 리스트에 즉시 반영됩니다. 전자결재 연동이 활성화된 경우 등록된 유형이 전자결재 양식의 선택 항목에도 반영됩니다.

삭제·수정 운영 규칙

연장근무 유형의 삭제·수정은 전자결재 연동 여부에 따라 처리 방식이 달라집니다.

전자결재 미사용 시

유형을 자유롭게 수정·삭제할 수 있습니다. 이미 승인된 신청 데이터가 있는 경우에도 삭제가 가능하지만, 기존 데이터의 유형 참조가 끊어질 수 있으므로 "미사용" 전환을 권장합니다.

전자결재 사용 시

진행 중인 결재 건이 있는 유형은 삭제할 수 없습니다. 결재가 완료되거나 반려된 후에만 삭제가 가능합니다. 수정 시에도 진행 중인 결재 건에는 영향을 주지 않으며, 이후 신규 신청부터 변경 내용이 적용됩니다.

삭제보다 사용여부를 "미사용"으로 전환하는 것이 안전합니다. 과거 근태 이력에서 유형 정보를 조회할 수 있도록 데이터를 유지할 수 있습니다.

외근·출장과의 차이점 요약

연장근무 유형 관리는 외근·출장 유형 관리와 화면 구조는 유사하지만, 아래 세 가지에서 근본적으로 다릅니다.

근태유형 선택

외근·출장은 근태 유형이 "외근" 또는 "출장"으로 고정됩니다. 연장근무는 연장근무 / 야간근무 / 휴일근무 중 하나를 선택할 수 있어 하나의 화면에서 세 가지 성격의 유형을 관리합니다.

근무시간 from~to

외근·출장은 총 근무 인정 시간을 시간+분 단위로 입력합니다. 연장근무는 시작 시각과 종료 시각을 from~to로 지정하여 실제 근무 구간을 시간대로 정의합니다.

출퇴근 인정 없음

외근·출장은 출/퇴근 인정 여부를 설정할 수 있지만, 연장근무는 정규 근무시간 이후의 추가 근무이므로 별도 출/퇴근 인정 설정이 존재하지 않습니다.
