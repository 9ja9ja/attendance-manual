출장 유형 관리

직원이 출장을 신청할 때 사용할 수 있는 출장 유형을 등록·수정·삭제하는 화면입니다. 등록된 출장 유형은 직원 신청 페이지와 전자결재 양식에서 선택 옵션으로 노출됩니다.

경로: 근태관리 > 근무유형 > 근태 유형 설정 > 출장 권한: 관리자
목차
화면 개요 검색 영역 유형 리스트 상세내용 주요 버튼 등록 팝업 필드 상세 삭제·수정 운영 규칙 외근과의 차이점 요약 연결 문서

화면 개요

출장 유형 관리 화면은 검색 영역 + 유형 리스트(좌) + 상세내용(우)으로 구성됩니다.

검색 영역

출장구분(출장명) 검색

유형 리스트 (좌)

등록된 출장 유형 목록 조회

상세내용 (우)

선택한 유형의 세부 설정 확인·수정

리스트에서 유형을 클릭하면 우측 상세내용 영역에 해당 유형의 설정값이 표시됩니다.

검색 영역

출장구분(출장명)을 기준으로 등록된 출장 유형을 검색합니다.

검색 조건 출장구분(출장명)을 입력합니다. 공란이 기본값입니다.
조회 실행 Enter 키 또는 조회 버튼을 클릭하여 검색을 실행합니다.

유형 리스트

등록된 출장 유형이 리스트 형태로 표시됩니다.

컬럼 설명
출장유형 근태유형 구분 (출장 고정)
출장구분(출장명) 관리자가 등록한 출장의 명칭
출장 인정 단위 일자 또는 기간
사용여부 사용 / 미사용

상세내용

리스트에서 출장 유형을 선택하면 우측에 표시되는 상세 설정 항목입니다.

근태유형 출장 고정 (disable 상태로 변경 불가)
출장구분(출장명) 출장 유형의 명칭을 입력합니다. 직원 신청 페이지와 전자결재 양식에서 선택 옵션으로 노출됩니다.

출장 인정 단위

출장 신청 시 인정되는 시간 단위를 설정합니다.

일자 (default)

하루 단위로만 출장을 신청합니다. 특정 날짜를 지정하여 1일 단위 출장을 등록합니다.

기간

시작일~종료일을 설정하여 기간 단위로 출장을 신청합니다. 여러 날에 걸친 출장에 적합합니다.

출/퇴근 인정

출장 시 출퇴근 기록을 근태에 어떻게 반영할지 설정합니다.

인정 (default)

등록한 출/퇴근 시간을 인정하여 근태 정보에 저장합니다.

미인정

등록한 출/퇴근 시간이 아닌, 신청 또는 전자결재 시 등록된 근무시간을 근태에 저장합니다.

근무 인정 시간

시간 8시간 (default). 0~12시간 중 선택합니다.
분 00분 (default). 10분 단위로 선택합니다 (00, 10, 20, 30, 40, 50).

휴게 인정 시간

시간 1시간 (default). 0~12시간 중 선택합니다.
분 00분 (default). 10분 단위로 선택합니다 (00, 10, 20, 30, 40, 50).
휴게 인정 시간은 근무 인정 시간을 초과하여 설정할 수 없습니다.

근무시간 자동인정

인정 (default)

설정된 근무 인정 시간이 자동으로 근태에 반영됩니다.

미인정

근무시간을 자동으로 인정하지 않습니다.

휴일 출장 인정

인정 (default)

출장 신청 시 휴일을 선택할 수 있습니다. 결재 처리 후 휴일 근무로 반영됩니다.

미인정

출장 신청 시 휴일을 선택할 수 없습니다.

연장/야간 근무 인정

인정 (default)

출장 중 연장·야간 근무가 발생할 경우 근태에 반영됩니다.

미인정

출장 중 연장·야간 근무를 근태에 반영하지 않습니다.

사용여부

사용 (default)

해당 출장 유형이 직원 신청 페이지와 전자결재 양식에 노출됩니다.

미사용

해당 출장 유형이 직원 신청 페이지와 전자결재 양식에서 숨겨집니다.

주요 버튼

등록 새로운 출장 유형을 등록하는 팝업을 엽니다.
수정 리스트에서 선택한 출장 유형의 상세내용을 수정 모드로 전환합니다.
삭제 리스트에서 선택한 출장 유형을 삭제합니다.
취소 수정 중인 내용을 취소하고 이전 상태로 되돌립니다.
저장 수정한 내용을 저장합니다.

등록 팝업 필드 상세

등록 버튼을 클릭하면 출장 유형 등록 팝업이 표시됩니다. 상세내용과 동일한 필드에 비고 항목이 추가됩니다.

근태유형 출장 고정 (disable)
출장구분(출장명) 명칭 입력 (필수)
출장 인정 단위 일자(default) / 기간
출/퇴근 인정 인정(default) / 미인정
근무 인정 시간 8시간 00분 (default), 시간 0~12 / 분 10분 단위
휴게 인정 시간 1시간 00분 (default), 시간 0~12 / 분 10분 단위. 근무 인정 시간 초과 불가
근무시간 자동인정 인정(default) / 미인정
휴일 출장 인정 인정(default) / 미인정
연장/야간 근무 인정 인정(default) / 미인정
사용여부 사용(default) / 미사용
비고 메모 입력 (최대 50자)
필수 입력: 출장구분(출장명)
중복 검증: 이미 등록된 출장구분(출장명)과 동일한 이름을 입력하면 "중복된 출장구분(출장명)입니다. 다른 출장구분을 사용해주세요." 메시지가 표시됩니다.

삭제·수정 운영 규칙

출장 유형을 삭제하거나 수정할 때는 전자결재 연동 여부에 따라 처리 방식이 달라집니다.

삭제 규칙

전자결재 미사용

해당 출장 유형을 사용한 기존 신청 건이 없으면 즉시 삭제됩니다. 기존 신청 건이 있으면 삭제할 수 없습니다.

전자결재 사용

해당 출장 유형으로 진행 중인 결재 건이 있으면 삭제할 수 없습니다. 모든 결재가 완료(승인/반려)된 후에만 삭제가 가능합니다.

수정 규칙

전자결재 미사용

상세내용의 모든 항목을 자유롭게 수정할 수 있습니다. 저장 시 즉시 반영됩니다.

전자결재 사용

해당 출장 유형으로 진행 중인 결재 건이 있으면 출장구분(출장명)을 변경할 수 없습니다. 나머지 항목은 수정 가능합니다.

진행 중인 결재 건이란 상신 후 아직 최종 승인 또는 반려되지 않은 건을 의미합니다.

외근과의 차이점 요약

출장과 외근은 유사한 구조를 가지고 있지만 아래 항목에서 차이가 있습니다.

항목 외근 출장
인정 단위 시간 / 종일 (2택) 일자 / 기간 (2택)
출/퇴근 인정 4택 (출근 인정·퇴근 인정·출퇴근 인정·미인정) 2택 (인정 / 미인정)
인정 단위 특성 시간: 시작~종료 시간 지정
종일: 하루 전체 외근		 일자: 하루 단위 출장
기간: 시작일~종료일 기간 출장
주요 용도 일과 중 일부 또는 전체를 외부에서 근무 1일 이상 타 지역으로 이동하여 업무 수행
외근은 시간 단위 관리가 핵심이고, 출장은 일자/기간 단위 관리가 핵심입니다. 출/퇴근 인정 역시 외근은 출근·퇴근을 개별 제어할 수 있지만, 출장은 전체 인정 또는 미인정으로만 운영합니다.
