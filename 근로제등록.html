근로제 등록·수정

신규 근로제를 생성하거나 기존 근로제를 수정하는 화면입니다. 두 개의 탭으로 구성됩니다 — 근로제 정보 탭에서 유형과 기본 근무 조건을, 근로제 정책 설정 탭에서 연장 근무 인정·법정 휴게·지각/조퇴 유예 등 운영 정책을 설정합니다.

화면 ID: HR-01-110-P / P2 진입: 근로제 관리 > 근로제 추가하기 탭 구성: 근로제 정보 + 근로제 정책 설정

탭 1 — 근로제 정보 공통 필드 표준(일반)근로제 시차근로제 선택근로제 스케줄근로제 유효성 검증 탭 2 — 근로제 정책 설정 연장 근무 인정 방식 법정 최소 휴게 시간 자동 반영 법정 최대 근로 시간 지각/조퇴 유예 시간 퇴근 기록 설정 간주 근로 여부 연결 문서

탭 1 — 근로제 정보

신규 근로제의 기본 정보를 작성하는 탭입니다. 사용 근로제 유형을 선택하면 해당 유형에 맞는 고유 필드가 추가로 노출됩니다.

공통 필드 (4가지 유형 모두 공유)

필드명 설명
근로제명 * 텍스트 입력 필드입니다. 동일한 근로제명이 이미 존재하면 저장 시 중복 오류가 발생합니다.
대표 아이콘을 통해 대표 근로제로 설정할 수 있으며, 대표 근로제는 신규 직원에게 자동으로 배포됩니다.
사용 근로제 * 표준근로제 · 시차근로제 · 선택근로제 · 스케줄근로제 중 선택합니다.
선택한 유형에 따라 하단의 고유 필드 영역이 분기됩니다.
1. 표준근로제 — 출퇴근 시간 및 근로 시간이 고정된 근무 유형
2. 시차근로제 — 출퇴근 시간은 조정 가능하나 근로 시간은 고정
3. 선택근로제 — 고정 출퇴근 없이 근무 주기 내 총 근로 시간 충족
4. 스케줄근로제 — 근무시간이 요일에 따라 변경될 수 있는 유형
소정 근무 요일 * 월~일 중 근무일을 선택합니다. 선택하지 않은 요일은 무급 휴일로 처리됩니다.
일반적으로 월~금을 선택하면 "월요일 ~ 금요일"로 표시됩니다.
주휴 요일 * 소정 근무 요일을 개근하는 경우 주휴 수당 산정에 사용되는 기준 요일입니다.
선택하지 않은 요일은 무급 휴일로 처리됩니다.
소정 근무 시간 * 드롭다운으로 단위를 선택합니다: 1일 / 1주(default) / 1개월
단위에 따라 자동 입력값이 설정됩니다 — 1일: 8시간 / 1주: 40시간 / 1개월: 209시간
키인으로 수정 가능합니다. 연장/야간/휴일 근로를 제외한 기준 시간입니다.
근무 패턴 근로제의 기본 규칙을 유지하면서 특정 주기마다 발생하는 예외 근무 사항을 등록합니다.
예: 월 1회 단축 근무, 토요일 특근 등
근무 패턴 추가 버튼 선택 시 근무 패턴 등록 팝업(HR-01-110-P3)이 열립니다.
출퇴근 방식 설정 * 실시간 등록 — 직원이 출/퇴근을 한 시점을 저장합니다.
자동 등록 — 근로제에 배포된 근무 시간을 자동으로 저장합니다.
실시간 등록 선택 시 추가 옵션:
· GPS를 기반으로 근무지 주소 출퇴근 등록
· WEB IP를 기반으로 근무지 주소 출퇴근 등록
· WIFI를 기반으로 출퇴근 등록
* 선택하지 않은 경우 별도의 통제 없이 직원의 출/퇴근 등록 시점으로 저장됩니다.

표준(일반)근로제 고유 필드

출퇴근 시간 및 근로 시간이 고정된 가장 기본적인 근무 유형입니다.

필드명설명
근무 시간 * 시작 시간(09:00 default) ~ 종료 시간(18:00 default)을 입력합니다.
입력된 총 시간이 우측에 자동 표시됩니다 (예: 9시간).
근무 인정 시간: 근무 시간 - 휴게 시간으로 자동 계산됩니다 (예: 8시간 00분).
근로제에서 인정하는 근무 시간만 근로 시간으로 인정됩니다.
휴게 시간 * 시작 시간(12:00 default) ~ 종료 시간(13:00 default)을 입력합니다.
휴게 인정 시간: 지정된 휴게 시간으로 자동 계산됩니다 (예: 1시간 00분).
법정 기준: 4시간 이상 근무 시 30분 이상, 8시간 이상 근무 시 1시간 이상 필수 부여.
기준 미달 시 저장 시점에 경고 얼럿이 노출됩니다.

시차근로제 고유 필드

출퇴근 시간은 조정 가능하나 소정 근로 시간은 고정되는 유형입니다. 시차 근무 리스트를 통해 출근 시간별 여러 세트를 관리합니다.

필드명설명
시차 근무 리스트 * 출근 시간별 근무 리스트를 버튼 형태로 관리합니다.
최초 진입 시 "9시 출근 근무" 버튼이 default로 노출됩니다 (표준근로제 초기값과 동일).
추가 버튼 선택 시 신규 근무 리스트가 생성됩니다.
버튼명 규칙:
· 추가 시 기본 명칭: "신규 근무" → 계속 추가 시 "신규 근무(1)", "신규 근무(2)" ...
· 기능 모음 > 근무 리스트 명칭 변경 시 사용자 지정 명칭으로 변경
· 출근 시간이 변경되더라도 지정된 명칭으로만 노출
리스트별 설정 각 시차 근무 리스트별로 소정 근무 시간, 근무 시간, 휴게 시간을 개별 설정할 수 있습니다.
리스트를 선택(버튼 클릭)하면 해당 리스트의 설정값이 하단 영역에 노출됩니다.
시차근로제 등록 버튼 선택 시 저장된 모든 시차 근무 리스트 항목이 개별 근로제로 등록됩니다.
생성 규칙: 근로제명_시차근무리스트명
예: "시차 근무제_9시 출근 근무", "시차 근무제_10시 출근 근무"

선택근로제 고유 필드

고정된 출퇴근 시간 없이 근무 주기 내 총 근로 시간을 충족시키는 유형입니다. 소정근무요일, 주휴요일, 근무패턴, 출퇴근 방식은 공통으로 공유합니다.

필드명설명
근무 주기 * 근무 시간을 정산하는 주기를 선택합니다.
1개월(default) / 1주 / 2주 / 3주 / 4주 선택 가능
선택한 주기에 따라 초과 근무 판단 기준이 달라집니다.
· 주 단위: 1주일 동안 40시간(최대 52시간) 기준으로 초과 근무를 판단합니다. 하루 8시간을 초과해도 주간 총량 내면 일반 근무로 인정됩니다.
· 1개월 단위: 고용노동부 가이드라인 기준, 1주일 40시간을 초과해도 일반 근무로 인정됩니다. 초과 근무 판단이 1개월 전체 근무 시간으로 적용됩니다.
근무 주기 시작일 * 근무 주기에 따라 노출값이 달라집니다:
· 1개월: "매월" + 드롭다운(1일 default ~ 말일)
· 주 단위: "첫번째" + 드롭다운(월요일 default ~ 일요일)
의무 근로 시간 * 근로자가 의무적으로 근로해야 하는 코어 타임을 설정합니다.
사용(default) / 사용 안함 선택
사용 선택 시 시간대 지정 컴포넌트가 노출됩니다 (11:00 ~ 16:00 default, 수정 가능).
사용 안함 선택 시 컴포넌트가 숨겨집니다.
표준 근무 시간 * 근로자와 협의된 표준 근무시간입니다. 09:00 ~ 18:00 default, 키인 가능.
입력된 근무 시간 - 휴게 시간이 우측에 자동 표시됩니다.
표준 휴게 시간 * 근로자와 협의된 휴게 시간입니다. 12:00 ~ 13:00 default, 키인 가능.
법정 기준: 4시간 근로 시 30분 이상, 8시간 근로 시 1시간 이상 필수 부여.
기준 미달 시 저장 시점에 경고 얼럿이 노출됩니다.
선택근무제 법정 근로시간 계산식
· 주 단위: 법정 근로시간 = 40시간 / 최대 = 52시간
· 1개월 단위: 법정 근로시간 = 40시간 × (정산기간 총 일수 ÷ 7일) / 최대 = 법정 + (12시간 × 정산기간 주수)

스케줄근로제 고유 필드

근무시간이 요일에 따라 변경될 수 있는 유형입니다. 교대 근무, 3교대 등에 사용됩니다. 설정된 스케줄이 자동 반복 적용됩니다.

필드명설명
스케줄 시작일 * 등록한 스케줄근로제의 기준이 되는 시작일입니다.
시스템 년월일이 자동 입력되며 키인 또는 달력 아이콘으로 변경 가능합니다.
스케줄 시작일부터 해당 스케줄이 자동 반복 적용됩니다.
근무 일수 * 스케줄을 지정할 근무 일수를 설정합니다. 5일 default, 최대 30일까지 입력 가능합니다.
설정한 일수만큼 일차별 스케줄 설정 테이블의 행이 생성됩니다.
지정된 근무 일수를 기준으로 스케줄이 자동 반복됩니다.
일차별 스케줄 설정 * 각 일차의 근무 스케줄을 설정하는 테이블입니다.
컬럼설명
일차 근무 일수에서 설정한 모든 일자가 순서대로 노출됩니다 (1일차, 2일차 ...)
업무 구분 드롭다운: 출근(default) / 휴일
· 출근 선택 시 → 근무 시간, 휴게 시간 활성화
· 휴일 선택 시 → 휴일 유형 드롭다운 노출 (무급휴일 default, 유급휴일, 주휴일)
근무 시간 출근일의 시작~종료 시간 (표준 근무 시간 default). 선택된 시간 - 휴게 시간이 우측에 자동 표시됩니다.
휴게 시간 출근일의 휴게 시작~종료 시간. 휴일인 경우 공란('-')으로 처리됩니다.
주휴일 중복 허용 * 스케줄 근무 시 7일 이하로 스케줄을 적용하면 1주에 주휴일이 2회 이상 발생할 수 있습니다.
허용 안함(default) — 주 2회 이상 주휴일 미허용. 2일 이상이어도 1일만 집계
허용 — 주 2회 이상 주휴일 허용. 모두 집계되어 계산
공휴일/약정휴일 근무 * 스케줄 편성 중간에 공휴일/약정휴일이 있을 때의 처리 방식입니다.
근무 안함(default) — 공휴일/약정휴일에는 스케줄 미적용, 다음 날부터 스케줄 재적용
근무 함 — 휴일 무시하고 기존 스케줄대로 근무 일정 유지
표준 근무 시간 * 근로 계약상 지정된 근무 시간. 09:00 ~ 18:00 default.
표준 휴게 시간 * 12:00 ~ 13:00 default. 법정 휴게 기준 동일 적용.
3교대 스케줄 예시
1순번: 출,출,출,출,휴,휴 (주간) / 2순번: 출,출,휴,휴,출,출 (야간) / 3순번: 휴,휴,출,출,출,출 (심야)
각 순번을 별도 스케줄근로제로 등록하고 직원별로 배포합니다.

탭 2 — 근로제 정책 설정

근로제의 운영 정책을 세부적으로 설정하는 탭입니다. 연장 근무 인정 방식, 법정 휴게/최대 근로 시간, 지각/조퇴 유예, 퇴근 기록, 간주 근로 여부를 관리합니다.

1. 연장 근무 인정 방식

근로제에 배포된 직원이 연장 근무를 하는 경우, 연장 근무 시간이 인정되는 방식을 선택합니다.

전자결재 승인 시간 인정

전자결재를 통해 신청한 연장 근무에 대해서만 근무 시간으로 인정합니다. 전자결재 서비스 미사용 회사는 비활성화됩니다.

연장 근무 시간 자동 인정

별도 인증 없이 지정 근무 시간 이상 근무 시 자동으로 연장 근무로 인정합니다.

미사용

연장 근무를 인정하지 않습니다. 근무 시간을 초과해도 연장 근로로 인정하지 않음. 포괄 임금제 등에서 사용합니다.

전자결재 승인 시간 인정 — 하위 옵션

선택 시 아래 3개 체크박스가 노출됩니다 (기본 모두 체크):

실근무 시간 적용 (최소 값) 전자결재 신청 연장 근무 시간과 실제 퇴근 시간 중 빠른 시간을 연장 근무로 인정하여 저장합니다.
연장 근무 사후 신청 허용 이전 일자에 진행한 연장 근무에 대해 사후 신청을 허용합니다.
전자결재 최종 승인 건 근로 시간 적용 전자결재가 최종 승인된 건에 대해서만 근로 시간으로 적용합니다.

연장 근무 시간 자동 인정 — 하위 옵션

선택 시 아래 옵션이 노출됩니다 (기본 모두 체크):

최소 연장 근무 시간 제한 연장 근무 시 최소 근무 시간을 설정합니다. 입력 필드 default '1', 숫자 2자리까지 입력. 단위: 시간(default) / 분 선택 가능.
연장 근무 단위 설정 연장 근무 시간을 몇 분 단위로 기록할지 설정합니다.
단위 미달 처리 설정한 단위에 미달하는 연장 근무 시간의 처리 방식을 지정합니다.
전자결재 미사용 회사의 경우 최초 진입 시: 전자결재 승인 시간 인정(disable) / 연장 근무 시간 자동 인정 / 미사용(default) 상태로 노출됩니다.

2. 법정 최소 휴게 시간 자동 반영

근로제 정보에서 설정한 소정 근무시간에 포함된 휴게시간 외, 연장 근무 시 추가로 부여되는 휴게시간을 설정합니다. 연장 근무 시간 자동 인정 선택 시에만 노출됩니다.

사용 (default)

아래 두 가지 방식 중 선택하여 연장 근무 시 추가 휴게 시간을 지정합니다.

미사용

연장 근무에 따른 추가 휴게 시간을 별도로 설정하지 않습니다. 하단 영역이 숨겨집니다.

지정된 시간으로 휴게 시간 지정 (default)

  • 하단에 시간 입력 필드가 고정 1개 노출됩니다 (18:00 ~ 19:00 default, 수정 가능).
  • 추가 버튼: 시간 입력 필드를 추가합니다. 최대 3개까지 추가 가능 (총 4개). 4개 초과 시 얼럿: "추가 휴게 시간은 4개까지 입력 가능합니다."
  • 삭제 버튼: 선택 시 해당 행이 별도 얼럿 없이 즉시 삭제됩니다.

4시간 마다 휴게 시간 지정

  • 입력 필드: default '30', 숫자 2자리까지 입력, 단위: 분
  • 4시간 연장 근무마다 설정한 분 단위의 휴게 시간이 자동 부여됩니다.

3. 법정 최대 근로 시간 설정 및 초과 기록 허용

사용 근로제에 맞는 법정 최대 근로 시간을 설정하고, 최대 근로 시간 초과 시 데이터 저장 여부를 선택합니다.

사용 (default)

주 52시간 default, 숫자 2자리 키인 가능. 최대 64시간까지 설정 가능합니다.
64시간은 재난·사고 등 예외 상황에만 적용되므로 확인 후 설정해야 합니다.

미사용

입력 필드가 disable 처리되며 52시간이 표시만 됩니다.

초과 기록 저장 방식

최대 근로 시간 초과 기록 저장 (default) 설정한 법정 최대 근로 시간을 초과하는 경우에도 실제 근무 시간 그대로 저장합니다.
최대 근로 시간 내 저장 초과하더라도 설정한 최대 시간까지만 저장합니다.

4. 지각/조퇴 처리 유예 시간 설정

설정된 근무 시간보다 직원의 출퇴근 시간이 다른 경우, 정상으로 인정해주는 시간 범위를 설정합니다. 지정한 인정 시간 내에서는 근무 시간에 포함됩니다.

사용

출근 인정 시간과 퇴근 인정 시간을 각각 설정합니다.

미사용 (default)

유예 시간을 별도로 설정하지 않습니다. 하단 옵션이 숨겨집니다.

사용 선택 시 하위 옵션

출근 인정 시간 default 10분, 숫자 최대 60까지 입력 가능. 단위: 시간 / 분(default) 선택 가능. 체크 해제 시 disable.
퇴근 인정 시간 default 10분, 숫자 최대 60까지 입력 가능. 단위: 시간 / 분(default) 선택 가능. 체크 해제 시 disable.

5. 퇴근 기록 설정

직원이 퇴근 기록을 누락하는 경우 퇴근 시간이 저장되는 방식을 설정합니다.

자동 퇴근 처리 (default)

근무 종료 시간, 또는 출근 시간으로부터 8시간 이후 자동으로 퇴근 처리됩니다.

출근 기록만 저장

퇴근 시간을 저장하지 않습니다. 추후 수정 요청을 통해 퇴근 시간 저장이 가능합니다.

6. 간주 근로 여부

사업장 밖에서 근로하여 실제 근로 시간을 정확히 산정하기 어려운 경우(재택, 장기간 출장 등) 근로한 것으로 인정할 수 있습니다.

사용

근로제 정보에서 설정한 근무시간을 모두 근무한 것으로 자동 반영합니다. 별도 출퇴근 기록 없이도 근무 인정됩니다.

미사용 (default)

저장된 출/퇴근 기록을 근무한 것으로 인정합니다.
