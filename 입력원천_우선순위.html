인정근로시간 — 통합 최적안

5개 기존 문서(통합로직 · 보완안 · 정상연장휴일 · 출장외근 · 월마감)와 입력원천 세부 우선순위를 하나로 통합한다.
"어떤 데이터가 → 어떤 순서로 → 어떤 역할로 → 인정근로시간에 도달하는가"를 한 문서에서 끝낸다.

2. 입력원천 우선순위 & 내부 순위

우선순위 = "높은 것이 낮은 것을 덮어쓴다"가 아니다.
각 데이터가 담당하는 역할이 다르며, 우선순위는 "어떤 데이터가 먼저 처리되어야 다음 데이터가 올바르게 작동하는가"를 정의한다.
근로제는 default, 근무패턴은 override. 패턴이 없는 날은 근로제 8h, 패턴이 있는 격주금요일은 패턴 4h가 적용된다.

3. 기준근무시간 산정 분기

근로제 × 근무패턴 조합에 따라 기준근무시간이 어떻게 결정되는지를 표로 정리한다.

패턴 상태기준근무시간예시 (근로제 9~18시)
패턴 미사용근로제 소정근로시간8h 그대로
패턴 사용 + 추가근무근로제 시간 + 패턴 시간8h + 패턴 2h = 10h (단, 초과분은 연장)
패턴 사용 + 단축근무근로제 시간 − 패턴 시간8h → 패턴 4h = 4h (격주금요일 오전근무)
패턴이 근로제보다 길 때 주의: 표준근로제에서 패턴 10h 설정 시, 초과 2h는 법적으로 연장근로(§50). "패턴 등록 시간" 기준으로 소정근로를 10h로 잡으면 연장 미산입 위험.
원칙: 패턴이 근로제보다 짧을 때만 "패턴 등록 시간" 기준이 안전.

출퇴근 방식계산 출발점인정시간 반영

5. 출장·외근 계산표

용어 통일: "출퇴근 인정"이라는 표현은 모호하므로 아래 용어를 사용한다.
• 근무시간기준: 설정된 소정근로시간을 그대로 인정 (예: 종일출장 = 8h)
• 신청시간기준: 실제 신청한 시간 범위를 인정 (예: 14:00~16:00 = 2h)

외근 인정시간 계산

설정가능 값인정시간 해석운영 포인트
인정 단위종일 / 시간종일: 하루 전체, 시간: 시간 단위 적용종일 외근 = 주로 근무시간기준
근무시간 자동 인정사용 / 미사용사용: 타각 없어도 소정시간 인정미사용 시 타각 필요
근무시간 계산 기준근무시간기준 / 신청시간기준근무시간기준: 소정=8h, 신청시간기준: 신청 범위핵심 분기
연장·야간 인정사용 / 미사용사용 시 소정 초과분 연장/야간 가산본시간 인정과 독립적 설정
휴일 근무 인정사용 / 미사용사용 시 휴일 외근도 인정본시간 인정과 독립적 설정

출장 인정시간 계산

설정가능 값인정시간 해석운영 포인트
근무시간 계산 기준근무시간기준 / 신청시간기준핵심 분기. 종일출장 vs 반일출장 판단기간 출장 시 일자별 적용
근무시간 자동 인정사용 / 미사용사용: 승인만으로 인정, 미사용: 타각 필요출장은 보통 자동 인정
연장·야간 인정사용 / 미사용사용 시 출장 기간 내 연장/야간 가산본시간과 독립적
휴일 근무 인정사용 / 미사용사용 시 출장 기간 내 휴일 인정본시간과 독립적
출퇴근 실적 연계있음 / 없음있음: 타각이 보조자료, 없음: 승인만 사용출장은 보통 "없음"
공통 원칙:
• 출장·외근은 타각이 없어도 인정 가능 — 승인이 1차 증거
• 본시간 인정, 연장·야간, 휴일은 각각 독립적 설정 — "본시간 인정 + 연장 미인정" 조합 가능
• 타각이 있으면 보완자료로 활용 (실제 근무 범위 확인용)

6. 핵심 산식

일별 기본 산식

교정실적구간 = 원시 타각 + 출퇴근 수정 승인
보완근무구간 = 출장·외근 간주근무로 채운 구간
대체구간 = 휴가·반차로 비근로 처리된 구간

판정대상시간 = 교정실적구간 + 보완근무구간 − 대체구간
인정근로시간 = 판정대상시간 − 휴게시간 − 제외시간 ± 관리자 보정

연장인정시간 = 인정근로시간 − 기준근무시간 (양수일 때만)
야간해당시간 = 인정근로시간 중 22:00~06:00 구간
휴일인정시간 = 휴일 판정일의 인정근로시간 전부

기준근무시간 산식

패턴 미사용: 기준근무시간 = 근로제 소정근로시간
패턴 사용: 기준근무시간 = 근로제 시간 ± 패턴 가감
출장·외근: 반영시간 = 유형설정 기준(근무시간기준/신청시간기준)
연장 자동: 연장 = max(0, 인정근로 − 기준근무시간)
전자결재: 연장 = min(승인시간, 실근무시간 − 기준근무시간)
결근: 인정근로시간 = 0 (근무일 + 미타각 + 미승인)

월 집계 산식

월 인정근로시간 = Σ 일별 인정근로시간
월 연장근로시간 = Σ 일별 연장인정시간
월 야간근로시간 = Σ 일별 야간해당시간
월 휴일근로시간 = Σ 일별 휴일인정시간
급여반영값 = 월마감 확정 결과 (원본 타각 아님)

7. 충돌 규칙 상세

대표 충돌 시나리오 10건

시나리오1차 자동처리최종 원칙
종일연차 + 타각 없음휴가 상태 그대로 확정정상 — 대체 완료
종일연차 + 짧은 타각경고 상태 생성관리자 확인 — 연차 취소 or 타각 무시
종일연차 + 장시간 타각충돌 상태 생성관리자 필수 보정 — 연차 취소 or 타각 제거
반차 + 미타각 (잔여 근무)근태 이상 상태관리자 보정 필요
반차 + 정상 타각 (잔여 근무)반차 시간 제외, 잔여 정상정상 — 대체+실적 공존
정상 근무일 + 미타각 + 기안 없음결근(0h) + 이상 상태관리자 보정 or 결근 확정
출장 + 미타각출장 유형 기준 인정정상 — 보완 적용
휴일 + 타각 있으나 미승인이상 상태 생성휴일근무 승인 없으면 미인정
휴무일 + 연차 신청오류 상태기안 접수 시 차단 — 휴무일에 연차 불필요
연차시간대 + 연장시간대 중복오류 상태기안 접수 시 시간대 중복 차단

8. 월마감 & 급여 반영

기본 원칙

  • 타각만으로 확정하지 않음
  • 근로제·패턴·승인·유형 함께 검토
  • 일별 분류 → 월마감은 집계·검토·확정
  • 급여는 원본 타각이 아닌 확정 결과 사용

기준데이터 4분류

  • 정상 출퇴근 — 프레임 안 정상 근무
  • 연장 인정 — 소정 초과, 정책 인정
  • 근태 이상 — 누락/지각/미승인 초과
  • 휴일 근무 — 휴일 판정일 인정시간

선택근로제 정산

  • 일별 연장 판정 보류
  • 정산기간(주/월) 종료 후 총합
  • 총실근로 > 법정기준 → 초과분 = 연장
  • 월마감 시 재정산 필수

월마감 상세 단계

단계설명실무 포인트
0. 대상 확정마감 대상 월·직원·사업장·조직월중 입퇴사자 고려
1. 기준 자료 확정캘린더, 근로제, 유형 고정마감 후 기준값 변경 영향 추적
2. 정합성 체크미승인 요청, 누락 타각 확인품질 게이트 — 통과해야 마감 진행
3~4. 근무일+프레임근무일/휴일 판정 + 소정근로 프레임"원래 어떻게 일하기로 되어 있었나"
4-A. 예상 출퇴근근무패턴 반영 직원별 예상 시간정상/이상 판정의 핵심 비교 기준
5~8. 실적+승인타각 정규화 → 교정 → 대체 → 보완STEP 4~7 일별 판정 그대로
9. 판정대상시간교정실적 + 보완근무 합산"회사가 인정할 후보 시간"
10~12. 차감+분해휴게 차감 → 인정시간 → 속성 분해연장·야간·휴일 분리 저장
13. 이상 판정이상 상태 + 기준데이터 4분류충돌/오류/미확정은 마감 차단
14~17. 정산+확정주간 정산 → 월 집계 → 보정 → 확정급여는 확정 결과만 사용

월 집계 결과 & 급여 연동

구분설명

10. 마감 전 체크리스트

항목확인 내용
미승인 요청모든 근태 요청(출퇴근수정, 연장, 휴가, 출장)이 승인 완료되었는가
누락 타각출근만 있고 퇴근 없는 날, 또는 그 반대가 없는가
충돌/오류/미확정 상태해당 상태가 모두 해소되었는가 (정상/경고만 마감 가능)
기준 자료 정합성근로제, 캘린더, 패턴이 해당 월에 올바르게 설정되어 있는가
유형 마스터 변경 영향휴가유형, 출장유형 변경이 과거 기간에 미치는 영향 확인
선택근로제 정산정산기간 단위 초과근무 재판정이 반영되었는가
관리자 보정 로그모든 보정에 사유, 보정자, 시각, 근거 기록이 있는가
급여 연동 준비확정 결과가 급여 시스템 입력 형식과 일치하는가
근로제 등록 → 근무패턴 관리 → 근태요청 관리 → 출장유형 관리 → 휴가유형 설정 → 집계·마감 →
