근무 패턴 관리

근로제의 기본 규칙을 유지하면서 특정 주기마다 발생하는 예외 근무 사항을 등록·수정·삭제하는 팝업 화면입니다. 월 1회 단축 근무, 토요일 특근, 격주 휴무 등 반복적인 근무 변형을 패턴으로 등록하여 자동 반영합니다.

화면 ID: HR-01-110-P3 진입: 근로제 등록 > 근무 패턴 추가 화면 유형: 팝업
목차
개요와 진입 동선 등록 필드 상세 반복 설정 상세 반복 종료 설정 근무 유형·정상 근로 인정·계산 기준 조회(상세) 화면 수정 화면 저장·삭제·취소 활용 예시 연결 문서

개요와 진입 동선

근무 패턴은 근로제 등록·수정 화면의 근무 패턴 추가 버튼을 통해 진입합니다. 등록된 근무 패턴은 근로제 정보 탭의 근무 패턴 필드에 목록으로 표시됩니다.

1. 패턴 추가 버튼

근로제 등록 화면에서 "근무 패턴 추가" 선택

2. 패턴 등록 팝업

패턴명, 반복 주기, 근무 시간, 유형 설정

3. 저장

패턴 목록에 등록, 근로제에 연결

4. 반복 적용

기준일부터 자동 반복 반영

등록 필드 상세

필드명설명
근무 패턴명 * 신규 생성하는 근무 패턴의 명칭을 입력합니다.
모든 텍스트 입력 가능 (한글, 영문, 숫자, 공란, 특수문자), 50자 이내.
예: "가족의 날 단축 근무", "토요일 특근", "매월 마지막 수요일 단축"
기준일 * 등록하는 근무 패턴이 적용되는 기준 일자입니다.
default: 시스템 년/월/일 + 1일. 키인 또는 달력 아이콘으로 변경 가능.
시스템 일자 포함 이전 일자는 입력/선택 불가능합니다.
저장된 일자부터 패턴이 적용됩니다.
근무 시간 * 패턴 적용일의 근무 시간을 입력합니다.
시작: 09:00(default), 종료: 18:00(default). 키인 가능.
입력된 근무 시간 - 휴게 시간이 우측에 자동 표시됩니다.
근무 인정 시간이 별도로 표시됩니다.
휴게 시간 * 패턴 적용일의 휴게 시간을 입력합니다.
시작: 12:00(default), 종료: 13:00(default). 키인 가능.
법정 기준: 4시간 근로 시 30분 이상, 8시간 근로 시 1시간 이상 필수 부여.
기준 미달 시 저장 시점에 경고 얼럿이 노출됩니다.
휴게 인정 시간이 별도로 표시됩니다.
비고 근무 패턴에 대한 설명을 작성할 수 있는 선택 필드입니다.
모든 텍스트 입력 가능, 100자 한도.
예: "직원 복지를 위한 매월 1시간 단축 근무"

반복 설정 상세

근무 패턴이 적용되는 반복 주기를 설정합니다. 반복 일정에 맞춰 근로제에 자동 반영됩니다.

반복 유형 설명 하위 옵션
없음 기준일 1회만 적용됩니다. 추가 옵션 없음
일 반복 주기에 따라 일별로 패턴이 적용됩니다. · 숫자 입력 필드: '1'(default), 2자리까지 입력 → "N일 마다 반복"
· 말일 체크박스: 선택 시 매월 말일에 반복 적용, 입력 영역 비활성화
주 반복 주기에 따라 주차별로 선택한 요일에 적용됩니다. · 숫자 입력 필드: '1'(default), 2자리까지 → "N주 마다 반복"
· 요일 선택 버튼: 월~일 중 반복할 요일을 활성화하여 지정
월 (default) 반복 주기에 따라 월별로 선택한 일자/주차 요일에 적용됩니다. · 숫자 입력 필드: '1'(default), 2자리까지 → "N개월 마다 반복"
· 드롭다운: 매월 N일 / N번째 X요일 / 마지막 X요일
예: "매월 24일", "네번째 목요일", "마지막 목요일"

반복 종료 설정

설정한 반복 주기를 종료하는 시점을 지정합니다. 반복이 종료된 이후부터는 근무 패턴이 반영되지 않으며, 반복 종료 시점을 다시 설정하여 패턴을 재적용할 수 있습니다.

종료 유형설명
없음 (default) 근무 패턴 종료 일자를 지정하지 않고 계속 반복합니다.
날짜 지정한 날짜에 패턴이 종료됩니다.
default: 시스템 년 +1, 월, 일. 키인 가능, 달력 아이콘으로 선택 가능.
회차 반복 횟수를 지정하고 해당 횟수까지 반복 후 패턴이 종료됩니다.
입력 필드: '1'(default), 2자리 숫자 입력 → "N회 반복"

근무 유형·정상 근로 인정·계산 기준

근무 패턴의 성격과 근태 집계 방식을 결정하는 3가지 핵심 설정입니다.

근무 유형

근무 패턴의 집계 유형을 선택합니다.

근무 (default)

등록 패턴을 근무(출근) 처리합니다. 근무 진행에 따른 근로 시간이 설정됩니다. 근무 집계 시 근무(출근)으로 저장됩니다.

휴무

등록 패턴을 휴일 처리합니다. 근무 집계 시에도 휴일로 저장됩니다.

정상 근로 인정

소정 근로에 포함 여부를 선택합니다. 근무 패턴의 근무 유형에 따른 근로 시간 인정 방식을 선택할 수 있습니다.

인정 (default)

기본 근로로 인정하여 소정 근로 시간(주 40시간)에 포함됩니다.

휴일 근무

기본 근로가 아닌 휴일 근무로 인정됩니다. 주 12시간 한도에 포함됩니다.

미인정

근로 시간으로 인정하지 않습니다. 급여 계산 대상에서 제외됩니다.

근무 시간 계산 기준

근태 기록에 적용되는 근무 시간 기준을 선택합니다.

근로제 표준 시간 (default)

출/퇴근의 경우 근무 패턴 등록 근무 시간을 적용하지만, 근태 기록은 근로제 정보에서 지정한 근로 시간을 적용합니다.

패턴 등록 시간

출/퇴근 뿐 아니라 근태 기록도 패턴에서 등록한 근로 시간을 적용합니다.

차이 예시: "가족의 날 단축 근무" 패턴에서 09:00~17:00으로 등록한 경우
· 근로제 표준 시간: 출퇴근은 09:00~17:00이지만 근태 기록은 근로제의 09:00~18:00 기준 적용
· 패턴 등록 시간: 출퇴근과 근태 기록 모두 09:00~17:00 기준 적용

조회(상세) 화면

저장된 근무 패턴에 대해 상세 정보를 조회할 수 있는 팝업입니다. 근무 패턴 목록에서 패턴을 선택하면 이 팝업이 열립니다.

  • 저장된 모든 정보가 disable 처리되어 조회 전용으로 노출됩니다.
  • 하단에 삭제, 취소, 수정 버튼이 제공됩니다.
  • 수정 버튼은 근로제 등록/수정 화면에서 진입한 경우에만 노출됩니다.

수정 화면

상세 화면에서 수정 버튼을 선택하면 저장된 근무 패턴을 불러온 등록/수정 팝업으로 이동합니다. 등록 화면과 동일한 폼에 기존 값이 로드됩니다.

수정 버튼은 근로제 등록/수정 화면에서 근무 패턴 관리 팝업에 진입한 경우에만 노출됩니다. 다른 경로에서 진입 시에는 조회 전용입니다.

저장·삭제·취소

저장

입력 사항을 저장하고 근무 패턴 목록에 반영합니다.

A004 "저장되었습니다." — 필수 정보(근무 패턴명 등)가 모두 입력된 경우
A010 "%필수 값%이 입력되지 않았습니다. 내용을 입력하고 다시 시도해주세요." — 필수값 미입력 시

삭제

조회된 근무 패턴을 삭제합니다. 삭제 시 컨펌 팝업을 거칩니다.

C004 "저장된 근무 패턴을 삭제하시겠습니까? 삭제된 근무 패턴은 복구할 수 없으며 사용 중인 경우 적용이 종료 처리됩니다."
A011 "삭제되었습니다." — 삭제 확인 후 성공 안내

취소

변경사항을 취소하고 팝업을 닫습니다.

  • 입력/수정된 정보가 있는 경우: 근무 패턴명 등 입력되거나 default 값에서 변경된 항목이 하나라도 있으면 컨펌 팝업이 노출됩니다.
  • 입력/수정된 정보가 없는 경우: default 값으로만 구성되어 있으면 별도 동작 없이 팝업이 종료됩니다.
C003 "변경된 내용이 저장되지 않습니다. 작성 중인 화면에서 나가시겠습니까?"

활용 예시

가족의 날 단축 근무

· 반복: 매월 / 1개월 마다 / 마지막 수요일
· 근무 시간: 09:00 ~ 17:00 (8시간 → 인정 7시간)
· 근무 유형: 근무
· 정상 근로 인정: 인정
· 계산 기준: 패턴 등록 시간
· 비고: "직원 복지를 위한 매월 1시간 단축 근무"

토요일 특근

· 반복: 매월 / 1개월 마다 / 첫번째 토요일
· 근무 시간: 09:00 ~ 13:00 (4시간)
· 근무 유형: 근무
· 정상 근로 인정: 휴일 근무
· 계산 기준: 패턴 등록 시간
· 비고: "월초 재고 실사를 위한 특근"

격주 금요일 휴무

· 반복: 매주 / 2주 마다 / 금요일
· 근무 유형: 휴무
· 정상 근로 인정: 미인정
· 비고: "격주 금요일 유연 휴무"

한시적 야간 연장

· 반복: 매일 / 1일 마다
· 반복 종료: 날짜 (2026-06-30)
· 근무 시간: 09:00 ~ 21:00
· 근무 유형: 근무
· 정상 근로 인정: 인정
· 계산 기준: 패턴 등록 시간
