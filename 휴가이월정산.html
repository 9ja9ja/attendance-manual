휴가 이월·정산

귀속년도의 잔여 유급연차를 정산 처리하고, 급여반영월에 연동하여 연차수당을 확정하는 화면입니다. 정산대상 선택, 정산기준 설정, 확정/확정취소를 관리합니다.

경로: 근태관리 > 휴가 > 휴가 이월·정산 권한: 관리자
목차
화면 개요 검색 영역 정산 목록 주요 버튼 대상자불러오기 팝업 정산기준설정 팝업 확정 처리 삭제 연결 문서

화면 개요

휴가 이월·정산 화면은 귀속년도가 종료된 후 잔여 연차를 정산(연차수당 전환)하거나 이월(차기연도 이관)하는 마감 업무를 처리합니다. 정산이 확정되면 근태마감 화면의 데이터와 연동되어 급여에 반영됩니다.

대상자 선택

대상자불러오기로 정산 대상 직원을 조회하고 선택합니다.

정산기준 설정

잔여 전체 정산 또는 일부 제외(이월) 등 기준을 설정합니다.

확정 처리

확정 시 근태마감 데이터에 연동되어 급여에 반영됩니다.

검색 영역

급여반영월 정산 결과가 반영될 급여월을 선택합니다. 기본값은 최신 급여반영월입니다.
연차적용기간 연차가 적용되는 기간을 조회 조건으로 설정합니다.
재직여부 전체(default) / 재직 / 퇴사 중 선택하여 필터링합니다.
사원 검색 사원명으로 검색합니다.

정산 목록

검색 조건에 해당하는 직원의 정산 현황을 테이블로 표시합니다.

기본 정보

사원코드 직원 고유 식별 코드입니다.
사원명 직원 이름입니다.
직급 현재 직급입니다.
부서 소속 부서명입니다.
재직여부 재직 또는 퇴사 상태를 표시합니다.
근속기간 N년 N개월 형식으로 표시됩니다.
연차시작일·종료일 해당 귀속년도의 연차 적용 시작일과 종료일입니다.

부여

발생 해당 귀속년도에 발생한 연차 총 개수입니다.
이월 전년도에서 이월된 연차 개수입니다.

소진

사용 결재완료된 연차 사용 건수입니다.
결재중 결재가 아직 완료되지 않은 건수입니다.

정산

일반 일반 정산(연차수당 전환) 처리된 개수입니다.
보상 보상 정산 처리된 개수입니다.
대체 대체 정산 처리된 개수입니다.

잔여

잔여 = 부여(발생 + 이월) - 소진(사용 + 결재중) - 정산(일반 + 보상 + 대체)

주요 버튼

확정처리 / 확정취소

드롭다운 버튼으로 확정처리 또는 확정취소를 선택합니다. 확정 시 근태마감 데이터에 연동됩니다.

대상자불러오기

정산 대상 직원을 조회하고 선택하는 팝업을 엽니다.

삭제

대기 상태의 정산 건만 삭제할 수 있습니다.

다운로드

현재 조회된 정산 목록을 엑셀 파일로 다운로드합니다.

대상자불러오기 팝업

정산 대상 직원을 조회하고 선택하는 팝업입니다.

귀속년도 정산 대상 귀속년도를 선택합니다.
연차사용기간 연차 사용 기간 범위를 설정합니다.
재직여부 전체 / 재직 / 퇴사 중 선택합니다.
정산대상연차 플러스연차(default)를 선택합니다. 잔여 연차가 있는 직원만 대상으로 조회됩니다.

조회된 대상 목록에서 직원을 선택한 후, 정산기준설정 팝업으로 이동합니다.

정산기준설정 팝업

대상자불러오기에서 선택한 직원에 대해 정산 기준을 설정합니다.

정산대상휴가개수 잔여전체(default): 잔여 연차를 모두 정산합니다.
일부제외: 1~99 범위에서 제외할 개수를 선택합니다. 제외된 분량은 차기연도로 이월됩니다.
이월기준일 이월 대상을 판단하는 기준일입니다. 이 날짜 이전에 부여된 연차만 이월 대상이 됩니다.
급여반영월 정산 결과가 반영될 급여월입니다. 근태마감 화면의 데이터와 연동됩니다.
확인 버튼 클릭 시 "정산 기준을 적용하시겠습니까?" 컨펌 얼럿이 표시됩니다. 확인하면 정산 목록에 대기 상태로 추가됩니다.

확정 처리

정산 목록에서 대상을 선택한 후 [확정처리] 버튼을 클릭하면 해당 건이 확정됩니다.

  • 확정 시 근태마감 대상 데이터에 연동되어 급여에 반영됩니다.
  • 확정취소는 드롭다운에서 선택하며, 확정된 건을 대기 상태로 되돌립니다.
확정된 정산 건은 근태마감 데이터에 즉시 반영됩니다. 확정 전 정산대상 및 급여반영월을 반드시 확인하십시오.

삭제

대기 상태의 정산 건만 삭제할 수 있습니다. 확정된 건은 삭제가 불가하며, 먼저 확정취소를 해야 합니다.

삭제된 직원은 대상자불러오기 팝업에서 다시 조회되어 재선택할 수 있습니다.
