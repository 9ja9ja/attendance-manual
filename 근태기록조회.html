근태 기록 조회

직원이 자신의 출퇴근 시간, 근태 신청 내역, 휴가 신청 내역, 근무 시간을 조회하는 화면들입니다. 4개 조회 화면(출퇴근 시간 조회, 근태 신청 조회, 휴가 신청 조회, 근무 시간 조회)을 하나의 매뉴얼로 정리합니다.

목차
화면 개요 출퇴근 시간 조회 근태 신청 조회 휴가 신청 조회 근무 시간 조회 연결 문서

화면 개요

근태 기록 조회는 직원이 자신의 근태 이력을 확인하기 위한 4개 조회 화면으로 구성됩니다. 각 화면은 독립적으로 접근할 수 있으며, 조회 목적에 따라 구분되어 있습니다.

출퇴근 시간 조회

일자별 출퇴근 시간, 근무시간, 근태상태 확인

근태 신청 조회

외근/출장/연장근무 등 근태 신청 내역 확인

휴가 신청 조회

연차/경조/출산육아 등 휴가 신청 내역 확인

근무 시간 조회

정상/연장/야간/휴일 근무시간 상세 확인

출퇴근 시간 조회

직원의 일자별 출퇴근 기록과 근태 상태를 조회하는 화면입니다.

기간 선택

  • 주간: 선택한 주의 월~일 출퇴근 기록을 조회합니다.
  • 월간: 선택한 월의 전체 출퇴근 기록을 조회합니다.

조회 항목

항목설명
일자근무일 날짜
출근시간출근 기록 시간
퇴근시간퇴근 기록 시간
근무시간출근~퇴근 사이의 실근무시간 (휴게시간 제외)
휴게시간근로제에서 지정된 휴게시간
근태상태정상출근/지각/조퇴/결근/미출근/미퇴근 등

출퇴근 수정 요청

출퇴근 기록이 누락되거나 오류가 있는 경우, 직원이 수정 요청을 할 수 있습니다.

  • 수정 사유를 입력하여 요청합니다.
  • 관리자 승인 후 출퇴근 기록이 수정됩니다.
출퇴근 수정 요청은 관리자 승인이 필수입니다. 직원이 직접 출퇴근 기록을 수정할 수 없습니다.

근태 신청 조회

직원이 신청한 근태(외근, 출장, 연장근무, 야간근무, 휴일근무, 간주근무) 내역을 조회하는 화면입니다.

기간 선택

조회할 기간의 시작일과 종료일을 선택하여 해당 범위 내의 근태 신청 내역을 조회합니다.

신청 유형별 필터

  • 전체 / 외근 / 출장 / 연장근무 / 야간근무 / 휴일근무 / 간주근무
  • 유형을 선택하면 해당 유형의 신청 건만 필터링되어 표시됩니다.

조회 리스트

항목설명
근태유형외근/출장/연장근무/야간근무/휴일근무/간주근무
구분유형 내 세부 구분 (예: 국내출장/해외출장)
신청일근태를 신청한 날짜
시작일~종료일근태 적용 기간
시간근태 적용 시간 (시작시간~종료시간)
상태진행중/승인/반려/취소
결재문서번호전자결재 연동 시 결재 문서 번호 (클릭 시 결재 문서로 이동)
전자결재 연동이 설정된 경우, 결재 진행 상태(진행중/승인/반려)가 함께 표시됩니다. 결재문서번호를 클릭하면 해당 결재 문서로 이동할 수 있습니다.

휴가 신청 조회

직원이 신청한 휴가 내역을 귀속년도 기준으로 조회하는 화면입니다.

귀속년도 기준 조회

귀속년도(YYYY)를 선택하여 해당 년도에 귀속되는 휴가 신청 내역을 조회합니다.

휴가유형별 필터

  • 전체 / 연차 / 경조 / 출산육아 / 기타
  • 유형을 선택하면 해당 유형의 휴가 신청 건만 필터링되어 표시됩니다.

조회 리스트

항목설명
휴가유형연차/경조/출산육아/기타
휴가명세부 휴가 명칭 (예: 결혼휴가, 배우자출산휴가)
신청일휴가를 신청한 날짜
사용기간휴가 시작일~종료일
사용일수사용한 휴가 일수 (0.5일 단위 가능)
결재상태진행중/승인/반려/취소

잔여 연차 현황

화면 상단 또는 하단에 해당 귀속년도의 연차 현황이 요약 표시됩니다.

총 부여 연차

기본연차 + 추가부여 연차

사용 연차

승인 완료된 사용 일수

잔여 연차

총 부여 - 사용 = 잔여

마이너스 연차가 허용된 경우, 잔여 연차가 음수(-)로 표시될 수 있습니다. 허용 한도는 관리자가 연차 정책에서 설정합니다.

근무 시간 조회

직원의 근무 시간을 유형별로 상세하게 조회하는 화면입니다.

기간 선택

  • 주간: 선택한 주의 일별 근무시간을 조회합니다.
  • 월간: 선택한 월의 일별 근무시간을 조회합니다.

일별 상세

항목설명
정상근무시간소정근로시간 내 실제 근무한 시간
연장근무시간소정근로시간 초과 근무 시간
야간근무시간22:00~06:00 사이 근무 시간
휴일근무시간휴일에 근무한 시간
총근무시간정상 + 연장 + 야간 + 휴일 합산

합계 행

조회 기간 내 각 근무 유형별 합계가 하단에 표시됩니다. 주간 조회 시 주간 합계, 월간 조회 시 월간 합계가 집계됩니다.

소정근로시간 대비 실근무시간 비교

배포받은 근로제의 소정근로시간과 실제 근무시간을 비교하여 표시합니다.

소정근로시간

근로제에서 정한 기간 내 총 소정근로시간

실근무시간

실제 출퇴근 기록 기반의 총 근무시간

소정근로시간 대비 실근무시간이 초과하면 연장근무로 분류됩니다. 선택근로제 등 유연근무제 적용 시에는 정산 기간 기준으로 비교됩니다.
