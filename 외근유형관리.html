외근 유형 관리

직원이 외근을 신청할 때 사용할 수 있는 외근 유형을 등록·수정·삭제하는 화면입니다. 등록된 외근 유형은 직원 신청 페이지와 전자결재 양식에서 선택 옵션으로 노출됩니다.

경로: 근태관리 > 근무유형 > 근태 유형 설정 > 외근 권한: 관리자
목차
화면 개요 검색 영역 유형 리스트 상세내용 주요 버튼 등록 팝업 필드 상세 삭제·수정 운영 규칙 마감 영향 연결 문서

화면 개요

외근 유형 관리 화면은 검색 영역, 유형 리스트(좌측), 상세내용(우측)의 3개 영역으로 구성됩니다.

검색 영역

외근 유형을 키워드로 검색합니다. 상단에 위치합니다.

유형 리스트 (좌)

등록된 외근 유형을 목록 형태로 보여줍니다. 항목을 선택하면 우측에 상세내용이 노출됩니다.

상세내용 (우)

선택한 외근 유형의 세부 설정을 확인하고 인라인으로 수정할 수 있는 영역입니다.

유형 리스트에서 항목을 선택하지 않으면 상세내용 영역은 비어 있는 상태로 표시됩니다.

검색 영역

외근 유형을 검색할 수 있는 영역입니다. 검색 조건은 다음과 같이 동작합니다.

  • 검색 필드는 공란이 기본값(default)이며, 공란 상태에서 조회 시 전체 외근 유형이 조회됩니다.
  • 키워드 입력 후 Enter 키 또는 조회 버튼을 눌러 검색을 실행합니다.
  • 검색 결과가 없으면 리스트 영역에 "조회된 결과가 없습니다" 메시지가 표시됩니다.

유형 리스트

등록된 외근 유형을 목록으로 보여주는 좌측 영역입니다. 등록된 순서대로 조회되며, 다음 컬럼으로 구성됩니다.

컬럼설명
근태유형외근으로 고정 표시됩니다.
외근구분(외근명)등록 시 입력한 외근 유형의 명칭입니다.
외근 인정 단위시간 또는 종일 단위로 표시됩니다.
근무시간해당 유형에 설정된 근무 인정 시간입니다.
사용여부사용 또는 미사용 상태를 표시합니다.
리스트에서 항목을 클릭하면 우측 상세내용 영역에 해당 유형의 세부 정보가 표시됩니다.

상세내용

유형 리스트에서 항목을 선택하면 우측에 노출되는 인라인 편집 영역입니다. 각 필드의 세부 구성은 아래와 같습니다.

근태유형 외근으로 고정 표시되며 수정할 수 없습니다(disable).
외근구분(외근명) 등록 시 저장된 외근 유형의 명칭입니다. 직원 신청 페이지와 전자결재 양식에서 외근 명으로 사용됩니다.
외근 인정 단위 시간 (default) / 종일
외근 근무를 시간 단위로 인정할지, 종일 단위로 인정할지 선택합니다.
출/퇴근 인정 미인정 (default) / 출근 인정 / 퇴근 인정 / 모두 인정
  • 미인정: 출퇴근을 직접 등록해야 합니다.
  • 출근 인정: 출근을 등록하지 않아도 자동으로 인정됩니다. 출근 시간부터 근무 시간이 인정됩니다.
  • 퇴근 인정: 퇴근을 등록하지 않아도 자동으로 인정됩니다. 퇴근 시간부터 이전 시간을 근무시간으로 인정합니다.
  • 모두 인정: 출퇴근 모두 자동으로 인정됩니다.
근무 인정 시간 시간 / 분으로 구성됩니다.
시간 단위: 시간 4h(default), 분 00(default)
종일 단위: 시간 8h(default), 분 00(default)
휴게 인정 시간 시간 / 분으로 구성됩니다.
시간 단위: 시간 0h(default), 분 30(default)
종일 단위: 시간 1h(default), 분 00(default)
근무 시간 자동 인정 인정 (default) / 미인정
인정으로 설정하면 별도의 직원 체크 없이 근무시간이 자동으로 저장됩니다.
휴일 근무 인정 휴일 포함 / 휴일 제외
휴일 포함 시 휴일에 외근한 근태가 인정됩니다.
연장/야간 근무 인정 인정 (default) / 미인정
인정으로 설정하면 8시간 초과 시 초과 시간이 연장·야간 근무로 반영됩니다.
사용여부 사용 (default) / 미사용
미사용으로 설정하면 직원 신청 화면에서 해당 외근 유형이 노출되지 않습니다.

주요 버튼

등록

새로운 외근 유형을 등록하는 팝업을 노출합니다. 팝업의 필드 구성은 상세내용과 동일합니다.

수정

상세내용 영역의 필드가 편집 가능 상태로 활성화됩니다. 하단에 취소/저장 버튼이 노출됩니다.

삭제

선택한 외근 유형을 삭제합니다. 전자결재 사용 여부에 따라 삭제 가능 여부가 분기됩니다.

취소 / 저장

  • 취소: 수정 상태에서 변경사항을 반영하지 않고 원래 값으로 복원합니다.
  • 저장: 변경사항을 저장합니다. 성공 시 안내 메시지가 표시됩니다.

등록 팝업 필드 상세

등록 버튼을 누르면 노출되는 팝업의 필드 구성입니다. 상세내용 영역과 동일한 구조이며, 추가로 비고 필드가 포함됩니다.

근태유형 외근으로 고정(disable)
외근구분(외근명) 필수 입력. 직원 신청 및 전자결재에서 사용될 외근 명칭입니다.
외근 인정 단위 시간(default) / 종일
출/퇴근 인정 미인정(default) / 출근 인정 / 퇴근 인정 / 모두 인정
근무 인정 시간 필수 입력. 시간 단위: 4h 00m(default), 종일 단위: 8h 00m(default)
휴게 인정 시간 시간 단위: 0h 30m(default), 종일 단위: 1h 00m(default)
근무 시간 자동 인정 인정(default) / 미인정
휴일 근무 인정 휴일 포함 / 휴일 제외
연장/야간 근무 인정 인정(default) / 미인정
사용여부 사용(default) / 미사용
비고 최대 50자까지 입력할 수 있는 메모 필드입니다.

등록 팝업 동작

  • 취소: 변경사항이 있으면 확인 팝업(컨펌)이 노출되고, 변경사항이 없으면 바로 닫힙니다.
  • 저장: 필수값이 모두 입력되었으면 저장이 완료됩니다. 필수값 미입력 시 에러 안내, 외근구분 중복 시 에러 안내가 표시됩니다.

삭제·수정 운영 규칙

외근 유형의 삭제 및 수정 시 아래 규칙이 적용됩니다.

수정

  • 리스트에서 항목을 선택한 뒤 수정 버튼을 누르면 상세내용 필드가 편집 가능 상태로 전환됩니다.
  • 취소 시 변경사항이 반영되지 않고 원래 값으로 복원됩니다.
  • 저장 성공 시 안내 메시지가 표시됩니다.
안내저장되었습니다.

삭제

  • 선택한 외근 유형이 전자결재에서 사용 중인 경우 삭제할 수 없으며, 별도 안내 메시지가 표시됩니다.
  • 여러 건을 선택하여 삭제할 때 일부만 전자결재에서 사용 중인 경우, 삭제 가능한 항목만 부분 삭제할 수 있는 컨펌 팝업이 표시됩니다.
  • 삭제 가능한 항목은 정상적으로 삭제됩니다.
안내삭제되었습니다.
안내전자결재에서 사용 중인 외근 유형은 삭제할 수 없습니다.
확인일부 항목이 전자결재에서 사용 중입니다. 삭제 가능한 항목만 삭제하시겠습니까.

마감 영향

외근 유형 설정은 인정근로시간 계산에 직접적인 영향을 미칩니다.

  • 외근 유형에 설정된 근무 인정 시간과 휴게 인정 시간이 일별 인정근로시간 산출의 기초 데이터로 사용됩니다.
  • 출/퇴근 인정 설정에 따라 출퇴근 기록 없이도 근무시간이 자동 반영될 수 있습니다.
  • 휴일 근무 인정 설정이 "휴일 포함"인 경우, 휴일 외근이 근태 데이터로 인정되어 월 마감 시 휴일근로 수당 산출에 반영됩니다.
  • 연장/야간 근무 인정이 "인정"인 경우, 8시간 초과분이 연장·야간 근로로 집계되어 월 마감 데이터에 포함됩니다.
외근 유형의 근무 인정 시간이나 인정 단위를 변경하면 이후 신청 건부터 변경된 기준이 적용됩니다. 기존에 승인된 건은 변경 전 기준이 유지됩니다.
