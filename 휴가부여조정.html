휴가 부여·조정

직원별 연차 발생·사용·잔여 현황을 조회하고, 수동으로 휴가를 부여하거나 조정(소진)하는 화면입니다. 전사 일정 등록과 엑셀 일괄 업로드도 이 화면에서 처리합니다.

경로: 근태관리 > 휴가 > 휴가 부여·조정 권한: 관리자

화면 개요

화면은 좌측의 직원 목록과 우측의 상세 영역(부여/소진 탭)으로 구성됩니다. 직원 목록에서 사원을 선택하면 해당 직원의 부여 이력 또는 소진 이력이 상세 영역에 표시됩니다.

직원 목록

귀속년도 기준 전체 직원의 부여·소진·잔여 현황을 한눈에 조회합니다.

상세 영역

선택한 직원의 부여 탭(발생 이력)과 소진 탭(사용 이력)을 전환하여 확인합니다.

검색 영역

귀속년도 조회 기준 연도를 선택합니다. 기본값은 당해 연도입니다.
사원명 사원명으로 검색합니다.
부서 부서를 선택하여 해당 부서 소속 직원만 필터링합니다.
직급 직급을 선택하여 필터링합니다.
입사일자 입사일자 범위를 지정하여 검색합니다.
조회 / 초기화 조회 버튼으로 검색을 실행하고, 초기화 버튼으로 모든 검색 조건을 기본값으로 되돌립니다.

직원 목록

검색 조건에 해당하는 직원의 연차 현황을 테이블로 표시합니다.

기본 정보 컬럼

사원명 사원대장에 등록된 이름입니다.
부서 소속 부서명입니다.
직급 현재 직급입니다.
입사일자 사원대장 기준 입사일자입니다.
근속기간 N년 N개월 형식으로 표시되며 실시간 업데이트됩니다.
공식: (현재연도 - 입사연도) x 12 + (현재월 - 입사월)

부여 컬럼

기본발생 시스템이 자동으로 생성한 연차 개수입니다.
가산 근속 가산에 의해 추가 부여된 연차입니다.
조정 관리자가 수동으로 부여한 연차입니다.
이월 전년도에서 이월된 연차입니다.

소진 컬럼

사용 결재완료된 유급연차 사용 건수입니다.
결재중 결재가 아직 완료되지 않은 건수입니다.
조정 관리자가 수동으로 소진 처리한 건수입니다.

잔여

잔여 = 부여 합계 - 소진 합계
부여(기본발생 + 가산 + 조정 + 이월) - 소진(사용 + 결재중 + 조정)

상세 영역 — 부여 탭

직원 목록에서 선택한 직원의 연차 부여 이력을 표시합니다.

발생일자 연차가 부여된 날짜입니다.
부여개수 해당 건으로 부여된 연차 개수입니다.
변경사유 부여 사유를 표시합니다. 시스템이 자동 생성한 연차의 경우 '자동생성' 문구가 노출됩니다.
자동생성 연차는 연차 자동부여 정책(기본설정)에 의해 시스템이 자동으로 생성한 건입니다. 수동 수정이 필요한 경우 휴가부여·조정 팝업을 사용하십시오.

상세 영역 — 소진 탭

선택한 직원의 연차 소진(사용) 이력을 표시합니다.

신청(조정)일 휴가를 신청한 날짜 또는 관리자가 조정한 날짜입니다.
사용기간 실제 휴가를 사용한 기간(시작일 ~ 종료일)입니다.
근태항목 휴가유형과 휴가구분을 함께 표시합니다.
사용일수 해당 건에서 소진된 연차 일수입니다.
사용내용 신청 시 입력한 사유 또는 내용입니다.
결재상태 결재 진행 상태(대기/진행중/완료/반려 등)를 표시합니다.
결재문서 연결된 전자결재 문서 링크입니다.

주요 버튼

다운로드

현재 조회된 직원 목록과 연차 현황을 엑셀 파일로 다운로드합니다.

업로드

엑셀 파일을 이용하여 휴가를 일괄 부여·조정합니다. 엑셀 업로드 팝업이 열립니다.

휴가부여·조정

직원 목록에서 선택한 직원에 대해 수동으로 휴가를 부여하거나 소진 처리합니다.

전사일정등록

전사 단위의 휴가 일정(창립기념일, 워크숍 등)을 일괄 등록합니다.

휴가부여·조정 팝업

직원 목록에서 대상 직원을 선택한 후 [휴가부여·조정] 버튼을 클릭하면 팝업이 열립니다.

사원 목록

선택된 직원 리스트가 표시되며, 각 직원의 잔여휴가일이 함께 노출됩니다.

조정 내용

조정기준 부여(default) / 소진 중 선택합니다. 부여는 연차를 추가하고, 소진은 연차를 차감합니다.
조정상세 드롭다운으로 선택합니다. 기본(default) / 가산 / 1년미만 / 보상 / 대체
유/무급구분 현재 유급 고정입니다. 향후 무급 옵션이 추가될 예정입니다.
휴가조정일 달력에서 날짜를 선택합니다.
부여/소진개수 숫자를 입력하며 소수점 4째 자리까지 허용됩니다. 조정기준이 소진인 경우 1개 고정(입력 불가)됩니다.
사유 조정 사유를 텍스트로 입력합니다.
저장 시 "휴가를 부여(또는 소진)하시겠습니까?" 확인 얼럿이 표시됩니다. 취소 시 변경사항 없이 팝업이 닫힙니다.
소진 처리 시 부여/소진개수는 1개로 고정되며 변경할 수 없습니다. 여러 건을 소진해야 하는 경우 건별로 처리하십시오.

엑셀 업로드 팝업

엑셀 파일을 이용하여 다수 직원의 휴가를 일괄 부여·조정할 수 있습니다.

파일찾기 업로드할 엑셀 파일(.xlsx)을 선택합니다.
등록 선택한 파일을 서버에 업로드하고 검증을 실행합니다.
양식다운로드 업로드용 엑셀 양식 파일을 다운로드합니다.
검증결과 업로드 후 성공 건수와 실패 건수가 표시됩니다.
실패내역 다운로드 검증 실패 건에 대한 상세 내역을 엑셀로 다운로드하여 오류 원인을 확인할 수 있습니다.
양식다운로드로 받은 템플릿의 컬럼 구조를 변경하지 마십시오. 컬럼이 다르면 검증 단계에서 실패합니다.
