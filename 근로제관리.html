근로제 관리

등록된 근로제를 조회·검색·삭제하고, 신규 근로제 생성과 일괄적용을 시작하는 목록 화면입니다. 이 화면에서 관리하는 근로제 데이터는 직원 배정, 예상 출퇴근 프레임 생성, 월마감 계산의 출발점이 됩니다.

화면 개요

근로제 관리 화면은 회사에 등록된 모든 근로제를 한눈에 조회하고, 신규 근로제를 추가하거나, 기존 근로제를 삭제·일괄적용하는 관리 화면입니다.

기준 데이터 조회

근로제명, 사용 근로제 유형, 근무요일, 소정근로시간, 사용 인원, 사용 여부를 목록으로 관리합니다. 이 값들은 직원별 근무 기준과 월마감 계산의 직접 입력값입니다.

운영 통제

대표 근로제는 삭제할 수 없고, 사용 인원이 있는 근로제는 삭제 전 재배정이 필요합니다. 미사용 전환 시 신규 배정만 차단되고 기존 직원은 유지됩니다. 단순 조회가 아니라 운영 통제 화면입니다.

검색 영역

화면 최초 진입 시 default 값으로 자동 조회되어 빈 화면 없이 현재 운영 중인 근로제 목록이 즉시 노출됩니다.

검색 조건 기본값 동작
근로제 유형 드롭다운 전체 전체, 일반, 시차, 자유, 스케줄 중 선택하여 유형별로 목록을 필터링합니다.
근로제명 검색 공란 입력 후 Enter 또는 조회 버튼 선택 시 LIKE %키워드% 부분일치 검색을 수행합니다. 공란인 경우 전체 검색됩니다.
조회 결과가 없는 경우 "조회된 데이터가 없습니다" 메시지가 노출됩니다. 조건 불일치와 미등록 상태를 명확히 구분할 수 있어야 합니다.

조회 목록

검색 결과는 아래 컬럼으로 구성된 그리드 목록으로 노출됩니다. row를 선택하면 해당 근로제의 상세 화면으로 이동합니다.

컬럼 내용 운영 포인트
구분 체크박스 영역입니다. 대표 근로제는 체크박스가 비활성화됩니다. 대표 근로제는 최상단에 고정 노출되며 삭제 대상에서 자동 제외됩니다.
근로제명 등록 시 입력한 근로제 명칭입니다. 직원 배정과 요청 승인에서 동일한 이름이 사용되므로 중복·모호성 관리가 중요합니다.
사용 근로제 일반근로제, 시차근로제, 자유근로제(선택근로제), 스케줄근로제 유형을 표시합니다. 월마감 계산에서 어떤 판정 기준을 적용하는지 결정하는 핵심 구분값입니다.
근무요일 등록 시 선택한 소정 근무 요일을 표시합니다. (예: 월, 화, 수, 목, 금) 휴일/근무일 판정의 기초입니다. 스케줄근로제의 경우 출/출/출/출/휴/휴 형태로 노출됩니다.
소정근로시간 1일/8시간, 1주일/40시간, 1개월/주 40시간 10분 등 단위와 함께 표시됩니다. 연장근로 판정의 기준선이며, 월 소정근로시간 계산의 입력값입니다.
사용 인원 해당 근로제에 현재 배정된 직원 수입니다. 삭제 가능 여부, 미사용 전환 정책 판단과 직접 연결됩니다.
사용 여부 사용/미사용 상태를 표시합니다. 미사용 시 신규 직원 배정이 차단되지만 기존 배정 직원은 유지됩니다.
대표 근로제 — 목록 최상단에 고정되며 체크박스가 비활성화됩니다. 신규 직원 입사 시 자동으로 배포되는 기본 근로제 역할을 합니다. 대표 근로제는 삭제할 수 없습니다.

주요 버튼

근로제 추가하기

선택 시 신규 근로제 생성 화면(HR-01-110-P)으로 이동합니다. 근로제 정보 탭에서 유형 선택 후 상세 설정을 진행합니다.

삭제

체크된 근로제를 삭제합니다. 사용 인원, 대표 근로제 여부, 배포 예정 참조에 따라 삭제 가능 여부가 달라집니다.

근로설정 일괄적용

선택한 근로제의 정책 설정 항목을 다른 근로제에 일괄로 복사 적용합니다. 반복 생성이 많은 조직에서 운영 효율을 높입니다.

삭제·미사용 운영 규칙

근로제 삭제는 단순 데이터 삭제가 아니라 직원 배정, 배포 예정, 대표 근로제 여부에 따른 운영 제약이 있습니다. 아래 상황별로 다른 얼럿/컨펌 메시지가 노출됩니다.

삭제 시 분기

상황 동작 얼럿 메시지
사용 인원이 있는 경우 삭제 불가
A001 "근로제를 사용하는 직원이 있어 삭제할 수 없습니다. 다른 근로제로 변경 후 삭제해주세요."
사용 인원이 없는 경우 삭제 가능 (컨펌 후)
C001 "선택한 N개 근로제를 삭제하시겠습니까? 저장된 근로제 내용이 모두 삭제됩니다."
일부만 삭제 가능한 경우 삭제 가능 항목만 대상으로 부분 삭제 컨펌
C002 삭제 가능한 항목만 삭제 대상으로 안내하는 부분 삭제 컨펌 팝업이 노출됩니다.
대표 근로제 포함 대표 근로제 삭제 불가
A002 대표 근로제는 시스템 기본 기준점이므로 삭제할 수 없습니다.
배포 예정 리스트에 포함 별도 경고
A003 배포 예정 리스트에 저장된 근로제이므로 별도 안내가 노출됩니다.

미사용 전환 규칙

사용 인원이 있는 근로제를 미사용으로 전환하면 기존 직원은 그대로 유지되지만 추가 직원을 배정할 수 없습니다. 이 경고는 별도 컨펌 팝업으로 안내됩니다.
  • 미사용으로 변경해도 이미 배정된 직원의 근로제는 변경되지 않습니다.
  • 신규 직원 배정 시 미사용 근로제는 선택 목록에서 제외됩니다.
  • 미사용 근로제를 다시 사용으로 변경하면 즉시 배정 가능 상태로 복원됩니다.

근로설정 일괄적용

선택한 근로제의 정책 설정 항목(연장 근무 인정 방식, 법정 최대 근로 시간, 지각/조퇴 유예 시간 등)을 다른 근로제에 일괄로 복사 적용하는 기능입니다.

적용 범위

근로제 정책 설정 탭의 항목들이 대상입니다. 근로제 정보 탭(근로제명, 유형, 근무요일 등)은 일괄적용 대상이 아닙니다.

활용 시나리오

동일한 연장 근무 정책·지각/조퇴 유예 시간을 여러 근로제에 적용해야 할 때 개별 수정 없이 한 번에 반영할 수 있습니다.

일괄적용 팝업에서는 근로제 정보 탭과 근로제 정책 설정 탭을 동일하게 조회할 수 있으며, 적용 버튼 선택 시 선택된 대상 근로제에 정책이 반영됩니다.

엑셀 다운로드·업로드

기능 설명
다운로드 현재 조회된 근로제 목록을 엑셀 파일로 내려받습니다. 조회 조건에 따라 다운로드 범위가 달라집니다.
업로드 (일괄 등록) 엑셀 양식에 맞춰 작성된 파일을 업로드하여 근로제를 일괄 등록합니다. 최초 업로드 시 양식 다운로드 안내가 제공됩니다.
